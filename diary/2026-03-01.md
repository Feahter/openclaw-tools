## 待办提醒 - 2026-03-02 21:30

### 问题：Xcode CommandLineTools 损坏

**问题描述**：
- xcrun 报错：`invalid active developer path (/Library/Developer/CommandLineTools), missing xcrun`
- 导致 OpenCode 无法正常运行
- gcc/clang 等编译器无法使用

**解决方案**：
```bash
# 1. 删除损坏的 CommandLineTools
sudo rm -rf /Library/Developer/CommandLineTools

# 2. 重新安装
xcode-select --install

# 或直接下载：
# https://developer.apple.com/download/all/?q=command%20line%20tools
```

**验证命令**：
```bash
xcrun --version
gcc --version
```
