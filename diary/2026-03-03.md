### 12. browser-use 原理与 Ollama 集成
**工作原理**：
用户指令 → AI模型(理解任务+规划步骤) → Playwright执行 → 返回结果

**需要 API Key 原因**：
- 默认使用 OpenAI GPT-4 / Anthropic Claude
- 云端付费 API

**Ollama 替代方案**：
- 理论上可用本地模型（Llama/Qwen）替代
- 需写适配代码解决 API 兼容性问题
- 短期成本高（开发适配代码），长期免费

### 13. Ollama RPA 模型分析
