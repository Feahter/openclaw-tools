# 短视频脚本模板库

## 001 逆袭类

### 标题：负债50万到年入100万，我只做对了这3件事

```
【开场】（0-3秒）
画面：主人公狼狈的样子
文案：曾经我负债50万，被人看不起

【困境】（3-10秒）
画面：被追债、被分手、被裁员
文案：失业、分手、抑郁症……人生跌入谷底

【转折】（10-20秒）
画面：遇到贵人/发现机会/开始行动
文案：直到我遇到……一切都改变了

【干货】（20-45秒）
画面：分享具体方法
文案：今天分享3个方法，第一……

【结果】（45-55秒）
画面：现在的成就
文案：现在我年入100万+，终于翻身

【引导】（55-60秒）
文案：想要同样逆袭的，评论区666
```
