## 晚间会话 2（23:44-23:53）

### 八字引擎核心 bug 修复（重要）

**根因**：五虎遁 offset 公式错误 + BASE 参考注释错误 + 时支公式遗漏 hour==0

**修复内容**：

#### 1. 日柱 BASE 纠正
- 原: `BASE_STEM=6(庚), BASE_BRANCH=5(辰)` → 2000-01-01=庚辰
- 改: `BASE_STEM=4(戊), BASE_BRANCH=6(午)` → 2000-01-01=戊午
- 依据: 万年历 2000-01-01 = 乙亥年十一月廿五**戊午**日

#### 2. 五虎遁 offset 公式（关键！）
- 原: `offset = year_stem * 2`
- 改: `offset = ((year_stem + 1) % 5) * 2`
- 口诀原文（百度百科）: 甲己之年**丙**作首（≠甲），丙辛**必定寻庚**起（≠戊）
- 验证: 丙年(stem=2)→offset=6→正月庚寅, 二月辛卯 ✓
- 修复前: 二月=己卯 ✗；修复后: 二月=辛卯 ✓

#### 3. 月柱传入参数修复
- 原: `get_month_stem_branch(year, 公历month)`
- 改: `get_month_stem_branch(year_stem, 农历lunar_month)`（先查农历月份）

#### 4. 时支公式修复
- 原: `if hour == 23: hour_index = 0`（仅处理 23 点）
- 改: `if hour == 0 or hour == 23: hour_index = 0`（0 点=子时也需修正）
- 原因: 23-1 点=子时, 1-3 点=丑时, 3-5 点=寅时...；0 点在子时范围内

#### 5. 五鼠遁验证
- 口诀: 丙辛从戊起（丙/辛日子时=戊）
- 验证: 丙申日, 寅时(3-5点)→时干=(丙*2+2)%10=6→庚 ✓
- canton.ai 确认: 寅时=庚寅, 卯时=辛卯 ✓

**验证结果**: 2026-03-23 05:00 北京时间(成都) → 真太阳时 3:56
- 修正前: 丙午 辛辰 戊未 乙辰 ✗
- **修正后: 丙午 辛卯 丙申 庚寅** ✓ (与 canton.ai / 万年历一致)
