# LLM Suite - LLM工具箱

整合4个LLM相关工具的完整套件。

## 子功能

### 1. llm (核心)
- 多提供商集成（OpenAI/Anthropic/Google/Ollama）
- 统一聊天接口
- 模型对比
- Token计数/成本估算

### 2. llm-app
- LLM应用开发
- Pathway框架集成

### 3. llmware
- 企业级RAG管道
- 小型专业化模型
- 知识库构建

### 4. llmware-ai
- AI驱动的RAG
- 向量搜索

## 使用方式

直接说需求：
- "用Claude回答" → llm
- "搭建知识库" → llmware
- "构建RAG应用" → llm-app
