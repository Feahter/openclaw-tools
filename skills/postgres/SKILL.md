# postgres

PostgreSQL 专用技能，支持高级查询、索引优化、JSON 操作。

## 功能

- 高级 SQL（窗口函数、CTE）
- JSON/JSONB 操作
- 全文搜索（FTS）
- 索引创建和优化
- VACUUM/ANALYZE
- 备份和恢复命令

## 使用方式

```bash
# 执行高级 SQL 查询
openclaw postgres query "SELECT ..."

# 管理索引
openclaw postgres index ...

# 执行 VACUUM/ANALYZE
openclaw postgres vacuum ...
```
