# SVG卡片生成器 (SVG Card Generator)

将任何文本内容转换为优雅简洁的SVG卡片，适用于展示、分享和可视化。

## 核心功能

- 多种预设模板
- 自定义尺寸和配色
- 支持中文排版
- 极简主义设计

## 卡片模板

### 模板1：经典文学 (Literary)
适用于：金句、小说摘录、诗歌
- 画布：480 x 760
- 配色：极简黑白
- 字体：KingHwa_OldSong (旧宋体)
- 布局：标题 | 分隔线 | 内容 | 分隔线 | 作者

### 模板2：逻辑分析 (Logic)
适用于：逻辑分析、思维导图、结构化内容
- 画布：640 x 1024
- 配色：极简主义
- 字体：KingHwa_OldSong
- 布局：标题 | 命题集 | 逻辑链 | 本质洞见

### 模板3：概念卡片 (Concept)
适用于：概念解释、学科介绍、知识卡片
- 画布：560 x 900
- 配色：极简主义
- 字体：KingHwa_OldSong
- 布局：标题 | 核心概念 | 详细内容 | 总结

### 模板4：微型卡片 (Micro)
适用于：一句话小说、金句、短文案
- 画布：480 x 320
- 配色：深色背景 (#000000)
- 主文字：#ffffff
- 次要文字：#00cc00
- 布局：标题 | 主题 | 内容

### 模板5：标准卡片 (Standard)
适用于：通用内容展示
- 画布：640 x 400
- 边距：30px
- 配色：极简主义
- 排版原则：对齐、重复、对比、亲密性

## 使用方法

### 基本用法
用户提供内容和模板选择：
```
用SVG卡片生成器，把这段文字做成[模板名]卡片：
[内容]
```

### 自定义配置
可以指定以下参数：
- **模板**：classic / logic / concept / micro / standard
- **画布尺寸**：宽 x 高
- **配色方案**：minimal / dark / warm / cool
- **标题**：卡片标题
- **作者/来源**：底部署名

## 排版原则

遵循四大设计原则：
1. **对齐**：元素间有明确的视觉联系
2. **重复**：重复使用设计元素，创造一致性
3. **对比**：通过大小、颜色、粗细创造层次
4. **亲密性**：相关元素靠近，不相关元素分离

## 配色方案

### Minimal (极简)
- 背景：#ffffff
- 主文字：#000000
- 次要文字：#666666
- 强调：#000000

### Dark (深色)
- 背景：#000000
- 主文字：#ffffff
- 次要文字：#00cc00
- 强调：#ffffff

### Warm (暖色)
- 背景：#faf8f5
- 主文字：#3d3d3d
- 次要文字：#8b7355
- 强调：#d4a574

### Cool (冷色)
- 背景：#f5f7fa
- 主文字：#2c3e50
- 次要文字：#5d6d7e
- 强调：#3498db

## 输出格式

生成SVG代码：
```svg
<svg width="[宽度]" height="[高度]" xmlns="http://www.w3.org/2000/svg">
  <!-- 背景 -->
  <rect width="100%" height="100%" fill="[背景色]"/>
  
  <!-- 标题 -->
  <text x="[x]" y="[y]" font-family="KingHwa_OldSong" font-size="[大小]" fill="[颜色]">
    [标题文本]
  </text>
  
  <!-- 分隔线 -->
  <line x1="[x1]" y1="[y1]" x2="[x2]" y2="[y2]" stroke="[颜色]" stroke-width="[宽度]"/>
  
  <!-- 内容 -->
  <text x="[x]" y="[y]" font-family="KingHwa_OldSong" font-size="[大小]" fill="[颜色]">
    [内容文本]
  </text>
  
  <!-- 作者信息 -->
  <text x="[x]" y="[y]" font-family="KingHwa_OldSong" font-size="[大小]" fill="[颜色]">
    [作者信息]
  </text>
</svg>
```

## 文本处理

### 自动换行
- 根据画布宽度自动计算换行位置
- 中文字符按字符数换行
- 英文按单词换行

### 文字布局
- 标题：顶部居中或左对齐
- 内容：居中对齐或左对齐，适当行距
- 作者：底部右对齐或居中

## 运行规则

1. 接收用户输入的内容和配置要求
2. 选择合适的模板或自定义配置
3. 生成符合排版原则的SVG代码
4. 输出完整的SVG代码，可直接渲染
5. 不添加额外解释，直接给出SVG代码

## 使用示例

**输入**：
```
用svg-card-generator把这句话做成文学卡片：
"一刻工夫，一碗肉已不见，骑手将嘴啃进酒碗里，一仰头，喉结猛一缩，又缓缓移下来，并不出长气，就喝汤。一时满屋都是喉咙响。"
标题：抽象之梯
作者：李继刚
```

**输出**：
生成对应模板配置的SVG代码
