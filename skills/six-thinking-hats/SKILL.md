# 六顶思考帽

一次戴一顶，思维更清晰。

## 快速指南

| 帽子 | 焦点 | 关键词 | 避免 |
|------|------|--------|------|
| ⚪ 白 | 事实 | 已知/未知 | 观点 |
| ❤️ 红 | 直觉 | 感受/情绪 | 解释 |
| ⚫ 黑 | 风险 | 问题/缺陷 | 乐观 |
| 💛 黄 | 价值 | 收益/机会 | 批判 |
| 💚 绿 | 创意 | 新想法 | 评判 |
| 🔵 蓝 | 流程 | 总结/行动 | 内容 |

## 快速模板

```
## [问题/决策]

### ⚪ 白帽子 - 事实
已知：[...]
未知：[...]

### ❤️ 红帽子 - 直觉
感受：[...]
直觉：[...]

### ⚫ 黑帽子 - 风险
风险：[...]
问题：[...]

### 💛 黄帽子 - 价值
收益：[...]
机会：[...]

### 💚 绿帽子 - 创意
新想法：[...]
替代方案：[...]

### 🔵 蓝帽子 - 行动
决定：[...]
下一步：[...]
```

## 时间分配

| 版本 | 时间 | 分配 |
|------|------|------|
| 完整版 | 30min | 每帽5min |
| 快速版 | 10min | 重点帽 |
| 极简版 | 5min | 白+蓝 |

## 场景变体

### 创意为主
跳过黑帽，绿帽延长时间

### 风险为主
跳过红帽，黑帽延长时间

### 快速决策
白 → 红 → 蓝（只关注关键信息）

## 运行规则

1. 一次只戴一顶
2. 按顺序使用（白→红→黑→黄→绿→蓝）
3. 每顶专注一个维度
4. 最后蓝帽总结
