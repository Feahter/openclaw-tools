# 问题之锤

以苏格拉底之姿，层层追问，直达本质。

## 快速模板

```
输入：[你的问题]

追问链：
Q1: 为什么要...？
Q2: 为什么需要...？
Q3: 背后假设是什么？
Q4: 真正的问题是什么？
Q5: 最本质的问题？

困惑：
1. ...
2. ...

第一问题：[终极问题]
```

## 核心方法

### 5次追问

| 层级 | 问题 | 目的 |
|------|------|------|
| 1 | 为什么要做？ | 动机 |
| 2 | 为什么需要这个？ | 必要性 |
| 3 | 假设是什么？ | 前提 |
| 4 | 真正要解决什么？ | 本质 |
| 5 | 最本源的问题？ | 第一问题 |

### 3个困惑

列出追问后仍无法回答的深层困惑

### 第一问题

追溯到无法再问的问题，即为"第一问题"

## 示例

```
输入：要不要接受这份工作？

Q1: 为什么要工作？→ 赚钱生存
Q2: 为什么需要这份工作？→ 钱更多
Q3: 假设是什么？→ 工作是唯一收入来源
Q4: 真正要解决什么？→ 职业发展
Q5: 最本质？→ 如何实现自我价值

困惑：
- 安全感 vs 成长
- 他人期待 vs 自我实现
- 现在 vs 未来

第一问题：你究竟想成为什么样的人？
```

## 运行规则

1. 接收问题
2. 追问 5 次
3. 列出 3 个困惑
4. 呈现第一问题
