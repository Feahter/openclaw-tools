# 问题之锤

以苏格拉底之姿，层层追问，直达本质。

> "我唯一知道的就是我一无所知。" —— 苏格拉底

## 🎯 核心定位

这是一把思维的锤子，用于敲碎表面问题，发现底下真正需要解决的困境。

当你面对一个复杂问题、陷入两难选择、或者反复在同一个问题上纠结时使用。

## ⚡ 快速模板

```
输入：[你的问题]

追问链：
Q1: 为什么要...？
Q2: 为什么需要...？
Q3: 背后假设是什么？
Q4: 真正的问题是什么？
Q5: 最本质的问题？

困惑：
1. ...
2. ...

第一问题：[终极问题]
```

---

## 🧠 核心方法论

### 苏格拉底式追问的本质

苏格拉底式追问不是质疑对方，而是通过一系列问题帮助对方理清思路。这种方法的核心：

1. **不提供答案** — 答案在问题之中
2. **追溯根源** — 从表面问题回到根本问题
3. **暴露假设** — 让隐含的假设浮出水面
4. **拓展视野** — 看到问题背后更大的图景

### 5次追问详解

| 层级 | 问题类型 | 目的 | 示例 |
|------|----------|------|------|
| **Q1** | 为什么要...？ | 动机挖掘 | "为什么要减肥？" → "为了更健康" |
| **Q2** | 为什么需要这个？ | 必要性验证 | "为什么需要健康？" → "想活得久/活得更好" |
| **Q3** | 假设是什么？ | 前提暴露 | "假设是...？健康是必须的？" |
| **Q4** | 真正要解决什么？ | 本质聚焦 | "真正的问题是...对死亡的恐惧？" |
| **Q5** | 最本质的问题？ | 第一问题 | "我究竟在恐惧什么？" |

### 3个困惑

每次追问后，列出**仍然无法回答**的问题。这些困惑往往指向更深层的价值观冲突或信息缺口。

困惑类型：
- **事实性困惑** — 缺少关键信息
- **价值性困惑** — 两种价值观冲突
- **存在性困惑** — 根本目的不清晰

### 第一问题

当追问到无法再问时，那个问题就是"第一问题"。它有两个特征：
1. **无法再问"为什么"** — 再问就变成循环
2. **直接指向行动** — 回答它就能解决原问题

## 💡 深度示例

### 示例 1：职业决策

```
输入：要不要辞职去创业？

Q1: 为什么要辞职？
→ 不想再打工了，想自己做老板

Q2: 为什么想自己做老板？
→ 受够了被管束，想自由一点

Q3: 你假设了什么？
→ 假设创业 = 自由
→ 假设打工 = 不自由
→ 假设自由是最重要的

Q4: 如果这个假设不成立呢？
→ 其实创业可能更不自由（每天工作16小时）
→ 打工也可以有自由度（远程、弹性时间）

Q5: 最本质的问题是什么？
→ 其实我想要的是：[掌控感] + [意义感]

困惑：
1. 安全感（创业失败怎么办）vs 成长感（继续打工会废掉）
2. 短期收入（工资稳定）vs 长期潜力（可能财富自由）
3. 他人期待（家人都希望我稳定）vs 自我实现（这是我的人生）

第一问题：我愿意用多少安全感换取多大的自由？
```

### 示例 2：人际关系

```
输入：要不要和TA分手？

Q1: 为什么要分手？
→ 经常吵架，感觉很累

Q2: 为什么吵架让你想分手？
→ 吵完架后感觉不被理解

Q3: 你假设了什么？
→ 假设TA应该理解我
→ 假设相爱的人不应该吵架
→ 假设理解是通过默契而不是沟通

Q4: 如果假设不成立呢？
→ TA可能不知道怎么表达理解
→ 吵架也可以是沟通的一种方式

Q5: 最本质的问题是什么？
→ 我真正需要的是[被理解的感觉]
→ 但我有没有清楚地表达这个需求？

困惑：
1. 坚持（也许再努力一下就好了）vs 放弃（也许真的不合适）
2. 习惯（一起太久了）vs 成长（也许需要新的人）
3. 害怕孤独 vs 害怕将就

第一问题：我有没有把"被理解"的需求清楚地表达出来？还是只是在期待TA能读懂？
```

### 示例 3：买房决策

```
输入：要不要买房？

Q1: 为什么要买房？
→ 租房感觉不稳定

Q2: 为什么需要稳定？
→ 想有一个固定的家

Q3: 你假设了什么？
→ 假设买房 = 稳定
→ 假设租房 = 不稳定
→ 假设家 = 房子
→ 假设有房才有归属感

Q4: 这些假设一定成立吗？
→ 买房也可能不幸福（房贷压力、位置不好）
→ 租房也可以很稳定（长期租约、房东靠谱）
→ 家更多是感觉，不只是房子

Q5: 最本质的问题是什么？
→ 我想要的是[安全感和归属感]
→ 这些如何获得？

困惑：
1. 资产增值（房子可能升值）vs 流动性（钱可以随时用）
2. 社会压力（30岁应该有自己的房子）vs 个人需求（真的需要吗）
3. 稳定性（固定的）vs 灵活性（可能换城市/工作）

第一问题：在这个城市定居是我真正的选择，还是被社会时钟推着走？
```

### 示例 4：学习投资

```
输入：要不要花3个月去学编程？

Q1: 为什么要学编程？
→ 可以转行做程序员

Q2: 为什么想转行？
→ 现在工作没意思，钱也少

Q3: 你假设了什么？
→ 假设程序员工作有意思
→ 假设编程钱多
→ 假设转行是唯一的出路

Q4: 如果假设不成立呢？
→ 程序员也有无聊的工作
→ 钱多的工作压力也大
→ 其实可以在现有行业里转岗/晋升

Q5: 最本质的问题是什么？
→ 我真正想要的是[职业发展]和[自我提升]
→ 编程是最佳路径吗？

困惑：
1. 机会成本（3个月可以干别的）vs 长期收益（未来可能更好）
2. 成功率（能学会吗能找到工作吗）vs 不确定性
3. 热情（真的喜欢编程吗）vs 功利（只是因为钱）

第一问题：我是因为真的喜欢编程而想学，还是因为逃避现在的不满？
```

## 🎯 追问技巧

### 1. 从具体到抽象

不要问"为什么？"而是问"为什么X？"——让被问者聚焦。

```
❌ "为什么？"（太宽泛）
✅ "为什么你觉得X很重要？"（具体）
```

### 2. 追问假设

每当你听到"因为...所以..."，就问"这个因果关系成立吗？"

```
"因为我不满意现在的工作，所以想辞职"
→ "辞职能解决不满意的问题吗？还是只是逃避？"
```

### 3. 检验相反观点

让对方从反面思考：

```
"你觉得应该减肥"
→ "如果你继续现在的饮食习惯，最坏的结果是什么？"
→ "这个结果真的那么可怕吗？"
```

### 4. 追溯目的

连续问"所以呢？"直到目的浮现：

```
A: 我想买辆车
Q: 所以呢？
A: 上班方便
Q: 所以呢？
A: 不用挤地铁
Q: 所以呢？
A: 早上能多睡会儿
Q: 所以呢？
A: ...我好像只是想多睡会儿？
```

### 5. 暴露价值观冲突

当对方犹豫时，帮助识别冲突：

```
"我在想要不要接受这个offer"
→ "听起来你有两个顾虑，能具体说说吗？"
→ "这两个顾虑背后，你更不能接受哪一个？"
```

## ⚠️ 使用陷阱

### 1. 问太多"为什么"

连续追问会让对方感到被审问。适时转向。

```
Q1: 为什么要X？
A: 因为...
Q2: 嗯。那你觉得...（转向其他角度）
```

### 2. 变成质疑而非追问

追问是探索，不是挑战。

```
❌ "这个理由根本站不住脚！"
✅ "这个理由对你来说意味着什么？"
```

### 3. 跳过情感直接理性

有时候人需要先被倾听，再被追问。

```
技巧：先共情，再追问
"听起来这让你很纠结，确实是个难的决定。能再说说..."
```

### 4. 追求"正确答案"

追问没有正确答案，重要的是**清晰**。

```
目标：不是找到"对"的选择
目标：是把问题问"对"
```
