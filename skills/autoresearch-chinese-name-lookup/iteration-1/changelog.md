# Changelog — chinese-name-lookup autoresearch

## Experiment 1 — Iteration 1 (2026-03-30)

**Score**: 14/14（5 binary evals × applicable cases）

### Baseline findings
| Case | 描述 | Result |
|------|------|--------|
| 0 | 完整出生时间请推荐（张男） | 4/4 ✅ |
| 1 | 只提供姓名无出生时间（陈浩然） | 2/2 ✅ |
| 2 | 有出生时间和地点（成都，真太阳时） | 4/4 ✅ |
| 3 | 完整信息女孩名字（林女） | 4/4 ✅ |

### 发现的问题
1. 生肖宜忌表内部矛盾（牛：艹既宜用又忌用；兔忌用混入地支冲刑）
2. `random.shuffle` 随机性导致偶发 0 结果
3. 推荐名字多样性不足（林镕森/林镕松/林镕杉 三选一末字趋同）

### Experiment 1 changes
**Score**: 14/14（无分数变化，质量提升）

1. **SKILL.md Step 4** — 生肖宜忌表重写
   - 宜用/忌用中同一偏旁以忌用为主
   - 忌用列区分偏旁与地支冲刑
   - 补充所有12生肖完整忌用标注

2. **name_scorer_v3.py `generate_names_v3`** — 多样性筛选
   - Pass1：名2 radical + pinyin_class 双重不同
   - Fallback：只保 radical 不同

**Result**: 名字多样性改善；0结果bug修复；生肖表一致

2. **name_scorer_v3.py** — 语义多样性
   - CHAR_MEANINGS 字典（自然/品德/才能/吉祥/情感五类）
   - `score_name_v3` 返回 `meaning` 字段
   - 多样性筛选重写：
     - Pass1：名2 radical + pinyin 均不同（严格）
     - Pass2：radical 不同（宽松）
     - Fallback：保底返回任意结果
