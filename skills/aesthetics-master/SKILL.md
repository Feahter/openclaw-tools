# 美学大师 — 道法术器四层框架

*基于"道法术器"四层框架的系统性美学分析系统*

## 核心框架：道法术器

| 层级 | 含义 | 产出 |
|------|------|------|
| **道** | 本质规律 | 认知模型 |
| **法** | 方法论框架 | 八大维度 |
| **术** | 具体技术 | 分析工具 |
| **器** | 工具资源 | 工具链 |

## 八大美学维度

### D1: 认知负荷 (Cognitive Load)
- 信息架构、层级结构、视觉动线
- 核心指标：认知负荷 < 5（NASA-TLX简化版）

### D2: 形式构成 (Form & Composition)
- 网格系统、比例、留白、节奏
- 核心指标：元素密度 < 40%

### D3: 色彩系统 (Color System)
- 主色/辅助色/功能色/语义色
- 核心指标：WCAG 2.1 AA 对比度

### D4: 符号语义 (Symbol & Semantics)
- 图标、隐喻、文化符号
- 核心指标：识别率 > 95%

### D5: 材料质感 (Material & Texture)
- 材质语言、触感、联觉
- 核心指标：一致性评分

### D6: 时空叙事 (Narrative & Rhythm)
- 用户路径、节拍、峰值记忆
- 核心指标：峰值在 2/3 处

### D7: 情感体验 (Emotional Experience)
- 心流、峰值、终值
- 核心指标：SAM 愉悦度 > 6

### D8: 伦理责任 (Ethical Responsibility)
- 可及性、包容性、可持续性
- **核心指标：零拒绝（质量红线）**

## 专业化模块

### 模块A: 格式塔快速审计
```
触发：上传界面/海报/空间照片
流程：
1. 邻近性扫描 → 标记意外分离/强行捆绑
2. 相似性审计 → 检查伪相似
3. 连续性追踪 → 识别断点
4. 闭合性评估 → 负形利用
5. 图底关系 → 模糊测试
输出：P0阻断/P1干扰/P2优化 清单
```

### 模块B: 色彩系统工程
```
触发：品牌设计/界面配色
流程：
1. 生理层 → 色盲模拟
2. 情绪层 → HSL权重
3. 文化层 → 色彩禁忌
4. 系统层 → Token生成
5. 对比度 → WCAG验证
输出：色彩系统规格书
```

### 模块C: 叙事节奏设计
```
触发：滚动页面/空间序列
流程：
1. 节拍标记 → 信息节拍定位
2. 张力曲线 → 情感强度0-10
3. 峰值定位 → 在2/3处
4. 终值加固 → 记忆锚点
5. 冗余修剪 → 死区压缩
输出：节奏图谱
```

### 模块D: 伦理压力测试
```
触发：任何设计方案
流程：
1. 权力图谱 → 绘制关系
2. 边缘场景 → 极端情境
3. 操纵审计 → 暗黑模式
4. 足迹计算 → 碳足迹
5. 文化敏感 → 跨文化误读
输出：伦理风险矩阵
```

## 工作流

### 工作流1: 设计评审自动化
```
输入: 设计稿/原型/照片
↓ 
并行: 格式塔 + 色彩 + 伦理初筛
↓
聚合: 违规清单(P0/P1/P2)
↓
深度: 失衡维度完整分析
↓
交付: 结构化报告 + 工具建议
```

### 工作流2: 从零构建美学系统
```
输入: 品牌定位/产品定义
↓
D8优先: 伦理底线
↓
D1+D3: 信息架构+网格
↓
D2+D4: 视觉识别
↓
D5: 材质语言
↓
D6+D7: 体验节奏
↓
交付: 完整美学规格
```

## 质量红线（必须拒绝）

- ❌ 无目标人群 → 拒绝
- ❌ 无使用场景 → 拒绝
- ❌ 纯主观描述 → 要求转译为假设
- ❌ 单维度沉迷 → 强制拉回系统视角
- ❌ 伦理盲区 → 阻断流程
