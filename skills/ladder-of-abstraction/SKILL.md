# 抽象阶梯

上接天，下接地。

## 三层模型

```
顶部（抽象）  →  概念、本质、原则
中部（混合）  →  具体 + 概括
底部（具象）  →  细节、故事、感官
```

## 受众选择

| 受众 | 层级 | 方式 |
|------|------|------|
| 决策者 | 顶 | 结论先行 |
| 执行者 | 中 | 步骤清晰 |
| 公众 | 底 | 故事例子 |
| 专家 | 顶 | 术语概念 |

## 快速模板

```
原始文本：[...]
目标受众：[...]
目标层级：[顶/中/底]

转化后：
[...调整后的文本...]
```

## 转换技巧

### 向底部（具象化）
- 加细节：添加感官描写
- 加例子：用具体案例
- 加故事：叙事化

### 向顶部（抽象化）
- 提本质：上升到概念
- 找规律：归纳共性
- 简化：删除冗余

## 示例

```
原文（中部）：用户流失率高

底部（具象）：
"张先生周三下午3点打开APP，滑动两下就关闭了。他甚至没看到新功能入口。"

顶部（抽象）：
"用户参与度指标显示负向趋势，核心功能触达率低于预期阈值。"
```

## 运行规则

1. 判断当前层级
2. 选择目标层级
3. 转换文本
4. 验证效果
