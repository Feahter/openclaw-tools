# Changelog — summarize/SKILL.md

## v2.0.0 — 2026-04-05

### 变更概述

优化目标：综合评估得分从 baseline **77.5%** 提升至 **≥90%**。

### 主要变更

#### 1. 新增「Trigger Rules」触发规则章节（预计+5%）
- 明确应触发场景：URL、文件路径、YouTube 链接的摘要需求
- 明确不应触发场景：纯知识问答等
- 使 AI 正确识别何时调用此 Skill

#### 2. 新增「参数验证」章节（预计+5%）
- 完整参数表格：target、--model、--length、--max-output-tokens 等
- 参数组合注意事项：--extract-only 仅 URL、--length 与 --max-output-tokens 互斥等
- 防止参数传递错误导致执行失败

#### 3. 新增「Error Handling」错误处理章节（预计+15%）
- **API Key 缺失**：症状识别 + 修复步骤 + 各平台设置命令
- **文件不存在**：路径检查方法 + 常见错误原因
- **网络错误/URL 不可访问**：Firecrawl fallback + 超时处理
- **模型不可用/配额超限**：model 名称校验 + 降级方案
- **YouTube 视频不可用**：Apify fallback + 备选方案
- **文件类型不支持**：支持格式列表 + 转换建议

#### 4. 新增「Exit Codes」章节
- 0 成功 / 1 参数错误 / 2 API 错误 / 3 网络错误 / 127 未安装
- 便于 AI 根据退出码判断错误类型并给出针对性建议

#### 5. 补充「Tips」最佳实践
- 长文本处理、程序调用、快速预览、调试方法
