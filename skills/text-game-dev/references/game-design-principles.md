# 游戏设计核心原则

## 核心循环

```
输入 → 反馈 → 奖励 → 动机强化
```

玩家每次操作都应该有明确的反馈和奖励。

## 趣味性来源

### 1. 挑战与技能匹配
- 难度曲线平滑上升
- 提供多种难度选择

### 2. 选择与后果
- 玩家能理解选择的影响
- 后果有累积效应

### 3. 叙事与情感
- 角色有深度和动机
- 情节有起伏和张力

### 4. 探索与发现
- 隐藏内容和秘密
- 多周目价值

## 文字游戏特殊考量

### 阅读体验
- 文本块不宜过长
- 重要信息视觉突出
- 对话与描述平衡

### 节奏控制
- 紧张与放松交替
- 信息密度有变化
- 悬念设置

### 代入感
- 第二人称"你"或第三人称
- 选择符合角色设定
- 避免打破第四面墙
