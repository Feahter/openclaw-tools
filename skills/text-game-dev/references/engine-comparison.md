# 游戏引擎详细对比

## Ren'Py

### 优点
- 功能最强大，社区活跃
- 支持Python扩展
- 跨平台（PC、移动、Web）
- 丰富的教程和资源

### 缺点
- 需要学习Python基础
- 界面相对朴素

### 适合项目
- 商业级视觉小说
- 复杂分支叙事
- 需要大量自定义功能

### 社区资源
- lemmasoft.forums
- GitHub大量开源项目

---

## TyranoBuilder

### 优点
- 可视化拖拽界面
- 无需编程
- 快速原型开发
- 素材市场丰富

### 缺点
- 付费版功能受限
- 自定义能力有限

### 适合项目
- 快速原型验证
- 简单视觉小说
- 无编程经验开发者

---

## VN Maker

### 优点
- 可视化操作
- 丰富素材库
- 中文支持好
- 社区活跃

### 缺点
- 付费软件
- 扩展性一般

### 适合项目
- 个人项目
- 商业小规模项目

---

## Twine

### 优点
- 纯浏览器运行
- 免费开源
- 专注叙事
- 导出HTML即可发布

### 缺点
- 无内置美术支持
- 适合纯文字

### 适合项目
- 交互小说
- 文字冒险
- 快速叙事原型

---

## GameCreator

### 优点
- 国产引擎
- 中文文档完整
- 可视化+脚本

### 缺点
- 社区相对小
- 国际化一般

### 适合项目
- 中文开发者
- 2D游戏
