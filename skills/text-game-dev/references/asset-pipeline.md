# 美术音效制作流程

## 角色立绘制作流程

### 1. 人设阶段
- 确定角色性格和外观特征
- 绘制草图（正面、侧面、背面）
- 确定配色方案

### 2. 线稿阶段
- 精细线稿
- 规划表情差分和姿势差分

### 3. 上色阶段
- 分层上色（皮肤、衣服、头发、配饰）
- 添加阴影和高光
- 输出透明背景PNG

### 差分清单（常见）
- 普通/微笑/大笑
- 生气/悲伤/惊讶
- 害羞/眨眼

---

## 背景图制作

### 类型
- 室内场景
- 室外场景
- 过渡场景（黑场/白场）

### 格式要求
- PNG/JPG/WEBP
- 分辨率匹配游戏分辨率
- 支持图层便于修改

---

## 音效制作

### 类型清单
| 类型 | 描述 |
|------|------|
| UI音效 | 按钮点击、菜单切换 |
| 环境音 | 风声、雨声、城市噪音 |
| 动作音 | 脚步、开关门 |
| 战斗音 | 攻击、受击、特效 |

### 技术规格
- 采样率：44.1kHz
- 格式：OGG（推荐）或 MP3
- 循环点标记（环境音）

---

## 音乐制作

### BGM清单规划
- 标题画面
- 日常场景
- 紧张场景
- 战斗场景
- 悲伤场景
- 结局

### 技术规格
- 时长：1-3分钟
- 无缝循环
- 淡入淡出点
- 格式：OGG或MP3
- 比特率：128-192kbps
