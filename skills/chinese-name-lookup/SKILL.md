# Chinese Name Fortune Analysis & Recommendation

基于**八字喜用神 + 生肖属相 + 十神十二长生**的传统中国取名体系。不使用五格剖象法（该方法为日本崛起创立，不适合中国人）。


## Step 1: 解析用户信息

**输入格式**（自然语言解析）：
- "帮宝宝起个名字，姓张，2025年3月15日上午10点出生，男孩"
- "姓陈名浩然怎么样"
- "我想要个女孩名字，姓林，预产期2026年6月"
- "2024年3月15日10点，姓张"

**提取字段**：
- `surname`: 姓氏
- `givenname`: 名（如果用户提供）
- `gender`: 性别（男/女，1=男，0=女）
- `birth_year`, `birth_month`, `birth_day`, `birth_hour`: 出生时间
- `birth_province`, `birth_city`: 出生地点

## Step 2: 本地八字排盘引擎

**完全本地化**，不依赖任何外部 API。包含：

### 2a. 农历转换 (`lunar_calendar.py`)
- 公历 → 农历（1900-2100年）
- 输出：农历年/月/日、是否闰月、地支纪年

### 2b. 生肖属相
- 根据出生年份判断：`(year - 4) % 12` 对应地支

### 2c. 八字天干地支 (`bazi_engine.py`)
- 年柱：`(year - 4) % 10` → 天干，`(year - 4) % 12` → 地支
- 月柱：五虎遁公式计算月干
- 日柱：JDN（Julian Day Number）算法
- 时柱：五鼠遁公式计算时干

### 2d. 十神系统
- 根据日干与其他干支的关系判定十神
- 映射：比肩/劫财/食神/伤官/偏财/正财/偏官/正官/偏印/正印

### 2e. 十二长生
- 日干在各柱的长生、沐浴、冠带、临官、帝旺等状态

## Step 5: 三维评分体系（V3）

**核心原则**：喜用神匹配 + 生肖宜忌 + 十神/长生互补

### 通过条件
1. 人格数不在凶数列表（仅作参考，不强制排除）
2. 总格数不在凶数列表（仅作参考）
3. 三才配置不为大凶

### 分数构成（满分 100）
| 维度 | 权重 | 说明 |
|------|------|------|
| 喜用神匹配 | 40% | 每个字匹配喜用神+12分 |
| 生肖宜忌 | 30% | 宜用偏旁+分，忌用偏旁-分 |
| 十神/长生补用 | 20% | 字本身或姓氏能补十神/长生状态 |
| 字义评分 | 10% | 常用好字加分 |

### 评分函数
```python
score_by_xiyongshen(name_chars, xiyongshen_list)  # 0-40
score_by_zodiac(name_chars, zodiac)                # 0-30
score_by_shishen(bazi_info, name_chars)             # 0-20
score_by_wuge(name_chars)                           # 0-10（参考）
```

## 错误处理

| 错误类型 | 处理方式 |
|---------|---------|
| API Key 无效/未配置 | 自动切换本地引擎，喜用神为估算值 |
| 笔画数表中无该字 | 使用偏旁推断，默认笔画数8 |
| 候选字不足3个 | 放宽生肖要求，直至凑满3个 |
| 农历转换超范围 | 提示超出1900-2100年范围 |

## 方法论说明

### 为什么不用五格剖象法？
五格剖象法（81数理）由**日本崛起（崛）**于20世纪初创立，民国时期才传入中国，并非传统中国取名方法。本 skill 以**八字喜用神**为核心，配合生肖属相和十神系统，更贴近真正的中国传统命理。

| 方法 | 权重 | 备注 |
|------|------|------|
| 八字喜用神 | ⭐⭐⭐⭐⭐ | 核心方法，中国传统 |
| 生肖属相 | ⭐⭐⭐⭐ | 传统方法，本地计算 |
