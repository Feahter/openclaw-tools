# 视角之镜

换个视角，让问题变简单。

## 5 种核心视角

| 视角 | 问题 | 适用 |
|------|------|------|
| **产品** | 用户需要什么？ | 体验、需求 |
| **技术** | 能否实现？ | 可行性 |
| **商业** | 成本收益？ | 投入回报 |
| **心理** | 人真正想要什么？ | 人性动机 |
| **AI** | 自动化能做什么？ | 效率预测 |

## 快速选择

```
问题类型 → 推荐视角

复杂问题     → 产品视角
创新需求     → 心理视角
可行性评估   → 技术视角
资源决策     → 商业视角
效率提升     → AI视角
```

## 视角模板

### 产品视角
- 用户痛点是什么？
- 解决后体验如何？
- 竞品怎么做？

### 技术视角
- 技术上可行吗？
- 需要什么能力？
- 风险点？

### 商业视角
- 投入多少？
- 产出多少？
- 周期多长？

### 心理视角
- 真正想要的是什么？
- 恐惧什么？
- 习惯什么？

### AI视角
- 哪些可自动化？
- 需要什么数据？
- 预测什么？

## 运行规则

1. 接收问题
2. 选择视角（根据场景）
3. 应用模板分析
4. 给出切入方案
