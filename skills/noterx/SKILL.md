# 薯医 NoteRx — 小红书笔记 AI 诊断

## 功能

输入小红书笔记（标题 + 正文 + 话题标签），返回：
- **CES 互动评分** — 小红书官方公式
- **6 维度 AI 评分** — 标题/内容/标签/互动/赛道/时效
- **爆款概率预测** — 带理由
- **Top3 优先改** — 具体可操作建议

---

## 使用方式

### 对话式（推荐）

直接粘贴笔记内容：

```
帮我看看这篇笔记：
标题：xxx
正文：xxx
#标签
```

或触发关键词：`薯医` / `爆款预测` / `这篇笔记能爆吗`

### CLI 模式

```bash
cd ~/.openclaw/workspace/skills/noterx
python3 scripts/noterx_engine.py --text '标题\n正文\n#标签'
```

## 示例对话

**用户**：
```
帮我看看这篇：程序员转行AI，三月学习路线分享
毕业后发现后端太卷了，决定转AI。有没有一起转行的搭子？
#AI转行 #程序员 #机器学习
```

**AI 回复**：
```
📋 薯医诊断报告

综合评分：52/100
标题有时效性+数字，但「三月」不够具体

| 维度 | 评分 | 建议 |
|------|------|------|
| 标题吸引力 | 58 | 改为「零基础3个月转AI岗」 |
| 内容利他性 | 45 | 补充周计划+资源清单 |
| 互动引导 | 72 | ✅ 搭子征集设计好 |
| 赛道竞争度 | 42 | 加「校招生」标签 |

爆款概率：15%
Top3：标题具体化 → 加周计划 → 加长尾标签
```

## CES 评分公式

```
CES = 点赞×1 + 收藏×1 + 评论×4 + 转发×4 + 关注×8
ER > 3% → 有爆款潜力
```

---

## 6 诊断维度

| 维度 | 评分依据 |
|------|---------|
| 标题吸引力 | 数字/情绪/悬念/场景 |
| 内容利他性 | 干货密度/可复制性 |
| 标签策略 | 长尾词/竞争度 |
| 互动引导 | 问句/征集/投票 |
| 赛道竞争度 | 差异化/细分切入 |
| 时效性 | 热点/节点/季节 |

## 已知限制

- 纯文本诊断无互动数据，CES 基于内容估算
- URL 模式需要小红书登录态，暂不可用
- 话题热度依赖 LLM 估算（非实时数据）
