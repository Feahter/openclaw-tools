# 逻辑之刃

拆解逻辑，验证推理。

## 快速模板

```
## 分析：[观点/决策]

### 核心命题
P1: [命题1]
P2: [命题2]
P3: [命题3]

### 推导链
P1 + P2 ⇒ P3
∴ P3 成立

### 结论
[是否自洽]
[逻辑漏洞]
```

## 核心方法

### 1. 提取命题
找出可判定真假的陈述句

### 2. 构建推导
```
A ∧ B ⇒ C
如果 A 且 B，则 C 成立
```

### 3. 验证链条
- 前提是否成立？
- 推导是否有效？
- 结论是否必然？

## 常见逻辑谬误

| 谬误 | 示例 |
|------|------|
| 因果错误 | A在B后，所以A导致B |
| 滑坡 | A⇒B⇒C⇒D，所以A发生D必发生 |
| 以偏概全 | 样本小就推广到全体 |
| 稻草人 | 曲解对方观点反驳 |

## 质量检查

- [ ] 命题可判定真假？
- [ ] 推导步骤无跳跃？
- [ ] 结论有据可依？
- [ ] 考虑了反例？

## 运行规则

1. 提取核心命题
2. 构建推导链
3. 验证逻辑
4. 给出结论
