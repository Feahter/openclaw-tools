# 📋 小说创作检查清单

> 基于「种子→萌芽→枝干→枝叶」四层架构 v4.0

---

## 🚀 创作前必读

| 层级 | 检查项 | 产出文件 |
|------|--------|---------|
| L1 | 新小说？ | 核心概念卡.md |
| L2 | 新卷开始？ | 大纲.md + 人物档案.md + 卷纲 |
| L3 | 新章节？ | 分镜脚本 + 章节规划 |
| L4 | 写作中？ | 正文 + 章节摘要 |

---

## 📝 L3 枝干层 - 分镜检查

### 章节类型判断

```
关键章（500字分镜）→ 死亡/牺牲/决战/表白/反转/高潮
普通章（350字分镜）→ 正常剧情推进
日常章（200字分镜）→ 铺垫/过渡/信息交代
```

### ✅ 必做项

- [ ] **分镜脚本** - 500字（关键）/ 350字（普通）/ 200字（日常）
- [ ] **衔接模板** - A时间/B场景/C因果 三选一
- [ ] **矛盾设计** - 核心冲突 + 困境 + 推动力 + 后果
- [ ] **爽点嵌入** - 至少1个爽点（关键章必填）

### 爽点类型参考

| 类型 | 示例 |
|------|------|
| 打脸 | 反派嘲讽 → 当众打脸 |
| 逆袭 | 被轻视 → 反杀 |
| 揭露 | 隐藏真相 → 当众揭开 |
| 情感 | 暗恋 → 表白成功 |
| 技术 | 难题 → 巧妙解决 |
| 反杀 | 绝境 → 反败为胜 |

## 🔗 L4 枝叶层 - 衔接检查

### QA 衔接5项必检

| # | 检查项 | 严重度 | 问题表现 |
|---|--------|--------|---------|
| 1 | 场景连续性 | 🔴 严重 | 突然换地图无过渡 |
| 2 | 时间连续性 | 🔴 严重 | 时间跳跃无过渡句 |
| 3 | 情绪连续性 | 🟠 中等 | 情绪180度转弯 |
| 4 | 因果连续性 | 🟠 中等 | 事件凭空出现 |
| 5 | 信息连续性 | 🟡 轻微 | 新人物无介绍 |

### 衔接修复模板

**问题：场景跳跃**
```
原文: 打完架后，他回到家。
修复: 打完架后浑身是血的陈默，踉跄着走出巷子。半小时后，他终于回到了家。
```

**问题：时间断裂**
```
原文: 第二天，陈默去找林清雅。
修复: 战斗结束后的第二天，伤口还在隐隐作痛。陈默拨通了林清雅的电话。
```

**问题：情绪断层**
```
原文: 听到这个消息，陈默很开心。他然后去吃饭了。
修复: 听到这个消息，陈默愣了一下，随即笑了。笑了三秒后，他深吸一口气，压下情绪。现在不是高兴的时候。
```

## 📊 章节质量评分

| 评分 | 等级 | 标准 |
|------|------|------|
| 90-100 | A | 6项全达标 |
| 75-89 | B | 5项达标 |
| 60-74 | C | 4项达标 |
| <60 | D | 需重写 |

### 评分项

- [ ] 字数：2000-3000字
- [ ] 爽点：至少1个爆点
- [ ] 衔接：与上一章自然衔接
- [ ] 人设：人物性格一致
- [ ] 润色：无错别字/病句
- [ ] 悬念：为下一章留钩子
