# ch036-求职信_3

## 三

星尘矿业公司，入职测试中心。

零重力测试区。

这是一个巨大的封闭空间，面积相当于一个标准的足球场。高高的天花板上是无数盏刺眼的LED灯，将整个房间照亮得如同白昼。房间的墙壁上覆盖着特殊的减震材料，能够承受各种高强度的冲击。

房间中央，是一个巨大的人工重力场发生器。

此刻，发生器正在运转，整个测试区域内处于零重力状态。

林启站在测试区域的入口处，看着眼前的景象，心情有些复杂。

零重力环境。

徒手挖掘一吨矿石。

三小时时限。

这他妈的到底是什么鬼测试？

"林启先生，"工作人员的声音从广播中响起，"请进入测试区域。测试将在三十秒后开始。”

"哦，"林启深吸一口气，迈步走进了零重力区域。

就在他踏入选定区域的那一刻，整个人突然失去了重量。

这种感觉......很奇怪。

就像是被一只无形的大手提到了半空中，身体完全不受控制。

林启的身体悬浮在离地半米高的位置，手脚都不知道该往哪里放。

"欢迎来到零重力矿洞，"一个机械声音响起，"您的任务是：在零重力环境下，徒手挖掘一吨矿石。时限：三小时。”

"等等，"林启举手，"在零重力环境下......怎么挖掘？”

作为一个从未经历过零重力环境的人来说，这个问题很合理。

"用这个，"工作人员递给他一把特殊的镐子。

镐子很小，看起来像是玩具。把手只有二十厘米长，镐头也只有成年人拳头大小。材质看起来像是某种合成金属，闪烁着淡淡的银光。

"......”

林启看着那把镐子，感觉自己像是在做梦。

零重力环境下，挖掘一吨矿石？

用这把玩具一样的镐子？

这他妈的到底是什么鬼测试？

"测试将在十秒后开始，"机械声音再次响起，"十、九、八、七......”

"我靠，"林启忍不住爆了粗口，"不带这么玩的......”

"三、二、一......开始！”

计时器开始倒计时。

三小时。

一百八十分钟。

六千四百八十秒。

林启深吸一口气，试图让自己冷静下来。

零重力环境下，挖掘确实比地球上容易——至少不用担心重力问题，碎矿石也会飘走不用担心砸脚。

但问题是......

矿石很硬。

非常硬。

林启的第一镐下去，只在矿石表面留下了一道白印。

"我靠......"林启忍不住再次爆粗口。

这到底是什么鬼矿石？

怎么会这么硬？

"您父亲的设定是......"张博士的声音在他脑海中响起，"矿石的硬度会根据挖掘者的'决心'而变化。”

"什么意思？"林启一边继续挖掘一边问道。

"意思是......如果你真的想挖，就能挖动。如果你的决心不够，矿石就会像铁一样硬。”

"......”

林启沉默了。

又是决心。

又是这种玄之又玄的东西。

他闭上眼睛，想起了父亲。

那个从未谋面的男人。

那个为了人类，主动与克苏鲁意识绑定的英雄。

他留下这段数据，是为了什么？

是为了让林启找到永恒之核。

是为了拯救人类。

是为了......

是为了让他成为一个合格的儿子。

一个合格的继承人。

一个合格的......英雄。

"给我动！！！”

林启大喝一声，一镐子下去。

矿石碎了。

不是一道白印。

而是彻底碎裂成无数小块。

"有效，"张博士的声音带着笑意，"继续。你看，决心够的话很容易吧？”

林启："......"

他看了看到处飘散的矿石碎片，又看了看手中的镐子。

这......也太神奇了吧？

难道这就是传说中的"心诚则灵"？

"别发呆了，"张博士催促道，"时间有限。赶紧继续。”

"哦，"林启继续挖掘。

一镐又一镐。

矿石一块又一块碎裂。

零重力环境下，挖掘确实比想象中要容易。没有了重力的束缚，碎矿石会自动飘走，不需要额外的清理工作。而且林启的每一镐都能准确地击中目标，因为身体可以自由移动。

但即便如此，这项工作仍然非常辛苦。

矿石虽然能碎，但需要的力量非常大。每一镐下去，林启都能感受到手臂传来的酸痛。

时间一分一秒地过去。

汗水从他的额头滴落，在零重力环境下形成一颗颗飘浮的水珠。

"呼......呼......"

林启喘着粗气，停下手中的动作。

两个小时过去了。

他不知道自己挖了多少，只知道身边的矿石堆越来越高。

"还有多少？"林启问道。

"大约还需要挖掘两百公斤，"张博士估算道，"加油，马上就完成了。”

"两百公斤......"林启咬咬牙，"行，继续。”

他深吸一口气，再次举起镐子。

又过了四十五分钟。

"好了！"张博士的声音响起，"恭喜你，任务完成！”

林启停下动作，看着眼前堆积如山的矿石碎片。

一吨。

整整一吨零重力矿石。

他做到了。

"恭喜，"工作人员的声音响起，"您通过了第一关。现在请跟我来，去参加第二关。”

---

零重力测试区的出口是一个类似气密舱的房间。

当舱门打开的那一刻，林启感觉自己重新回到了正常重力环境。

那种脚踏实地的感觉......真好。

"林启先生，"工作人员微笑着说，"请跟我来。第二关是智力测试，地点在七楼。”

"好的，"林启点点头，跟随工作人员走进了电梯。

电梯一路向上，最终停在了第七层。

这一层的设计风格与测试区完全不同。

走廊两侧是无数个独立的房间，每个房间的门上都标着不同的数字。房间的墙壁是透明的玻璃，可以清楚地看到里面的情况。

林启好奇地看向其中一个房间。

房间里坐着一个年轻人，正对着面前的光屏发呆。年轻人的表情非常痛苦，显然是被题目难住了。

"这里是智力测试区，"工作人员解释道，"每个房间都有不同的题目。根据您选择的题目难度，会有不同的评分。”

"评分？"林启皱眉，"这个测试还有评分？”

"当然，"工作人员说，"您父亲设定的规则是：题目越难，通过后的奖励越高。”

"奖励？"林启的眼睛亮了起来，"什么奖励？”

"这个嘛......"工作人员露出一个神秘的微笑，"等您通过后就知道了。”

"好吧，"林启耸耸肩，"那......给我来最难的题目。”

"最难的？"工作人员愣了一下，"您确定？”

"确定，"林启点点头，"既然来了，当然要挑战最高难度。”

"好，"工作人员的嘴角露出一丝笑意，"请跟我来。”

---

工作人员带着林启来到了走廊最深处的房间。

房间的门是深蓝色的，与其他房间的白色形成了鲜明的对比。

门上刻着一行字：

"终极挑战"

"这是......"林启看着那行字，"最难的题目？”

"对，"工作人员说，"这一关只有一道题。但这道题的难度......是所有题目中最难的。”

"多难？”

"曾经有人尝试过，"工作人员的表情变得严肃，"但至今没有人能答对。”

"......”

林启沉默了。

连一个人都没有答对的题目？

那到底有多难？

"您确定要尝试吗？"工作人员问道，"如果选择放弃，可以直接通过这一关，获得基础奖励。”

"不，"林启摇摇头，"我要挑战。”

"好，"工作人员打开门，"请进。”

---

房间很小，只有十平方米左右。

房间中央放着一张桌子，桌子上有一个发光的屏幕。

屏幕上面显示着一道题。

但看到题目的那一刻，林启整个人都愣住了。

题目是：

"请证明：1+1=2”

"......”

林启看着屏幕，脑子里一片空白。

这道题......

就是传说中的"终极挑战"？

1+1=2？

这他妈的还需要证明？

"这......"林启嘴角抽搐，"这就是最难的题目？”

"对，"张博士的声音响起，带着明显的笑意，"你父亲......真的很会玩。”

"......”

林启盯着屏幕，陷入了沉思。

1+1=2。

这是最基本的数学公理。

在皮亚诺公理体系中，1+1=2确实需要证明。但这种证明......

太长了。

而且太复杂了。

林启回忆了一下自己学过的数学知识。

皮亚诺公理......自然数定义......加法定义......

如果真的要严格证明1+1=2，可能需要数百行的推导过程。

但这显然不是父亲想要的答案。

因为......

如果只是要一个数学证明，根本不需要设置成"终极挑战"。

一定有其他的含义。

林启盯着屏幕。

突然，他注意到了题目的一个细节。

题目是：请证明：1+1=2

但注意看冒号的位置。

"请证明：1+1=2"

这个冒号......

是不是在暗示什么？

林启的大脑飞速运转。

1+1=2......

在不同的语境下，这个等式有不同的含义。

数学上：1个+1个=2个

生活上：1个人+1个人=2个人

爱情上：1颗心+1颗颗心

心=2但这些都不是重点。

重点是......

父亲为什么要设置这道题？

他想要考验什么？

林启闭上眼睛。

他想起了之前在星尘矿业测试中，张博士说过的话。

"你父亲说过......真相往往藏在细节里。”

细节......

林启重新睁开眼睛。

注意到题目的格式："请证明：1+1=2"

注意那个冒号。

在编程语言中，冒号通常用于......

定义！

比如说，在Python中：

```
def add(a, b):
    return a + b
```

又或者说，在JSON中：

```
"key": "value"
```

那么......

林启的眼睛亮了起来。

"请证明：1+1=2"

父亲想要的，不是数学证明。

而是......

一个关于"定义"的证明。

1是什么？

2是什么？

加法是什么？

如果不能定义这些，就不能证明1+1=2。

所以......

林启在光屏上输入：

"答案：1+1=2是一个公理，不需要证明。在皮亚诺公理体系中，1被定义为0的后继，2被定义为1的后继，加法被定义为后继运算的拓展。因此，1+1=2不是被'证明'的，而是被'定义'的。”

"但更深层的含义是：这道题本身就是一个陷阱。”

"因为我父亲设置这道题的目的，不是要一个数学答案。”

"而是要告诉我们：有些事情，不需要证明。”

"比如说......”

"爱。”

"比如说......”

"信念。”

"比如说......”

"守护。”

"这些，都不需要证明。它们只需要被相信，被践行。”

"而这......就是我父亲想让我明白的真相。”

屏幕沉默了很久。

然后，屏幕上跳出一行字：

"回答正确。恭喜您，通过了终极挑战。”

---

房间的门打开了。

工作人员站在门口，脸上带着敬佩的表情。

"恭喜您，"工作人员说，"您是第一个通过终极挑战的人。”

"所以......"林启问道，"我通过了？”

"通过了，"工作人员说，"而且，您获得了一份特殊的奖励。”

"什么奖励？”

"这个，"工作人员递给林启一张金色的卡片，"这是您父亲留给您的。”

卡片上印着一行字：

"答案，在虚空中漂流。”

"又是提示......"林启看着卡片，陷入了沉思。

不在星尘里......

在虚空中漂流......

这到底是什么意思？

"林启先生，"工作人员说，"您已经通过了星尘矿业的所有测试。现在请跟我来，去参加综合素质测试。”

"综合素质测试？"林启问道，"那是什么？”

"这个嘛......"工作人员露出一个神秘的微笑，"等您到了就知道了。”
