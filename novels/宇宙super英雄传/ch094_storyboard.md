# ch094 分镜脚本

## 场景1：最终题降临（开场）
- **镜头**：全屏显示最终题题目，红色倒计时300秒
- **氛围**：紧张、压抑
- **重点**：苏小暖揭示这是"开放题"而非选择题，难度暴增
- **弹幕**：恐慌、吐槽、求老林出手

## 场景2：林启的沉思（转折）
- **镜头**：林启闭眼回忆父亲遗言，与苏小暖对话拆解题目
- **核心矛盾**：瑶瑶传承知识揭示"最终题没有正确答案，只有真实的答案"
- **氛围**：冷静中暗藏紧张

## 场景3：逆向引导（高潮前奏）
- **镜头**：林启面对镜头宣布"不给答案"
- **反转**：不引导统一选择，而是让每个人说真话
- **弹幕**：从震惊到理解
- **金句**："如果连坚持自己的信念都做不到，那我守护的这个宇宙，还有什么意义？"

## 场景4：三十亿个"坚持"（高潮）
- **镜头**：全宇宙弹幕汇聚，"坚持"二字如雪崩般涌来
- **永恒之核**：感应三十亿信念共振
- **苏小暖**：泪流满面报出一致性指数 97.3%
- **音乐**：从沉默到交响乐级宏大

## 场景5：审判通过（情感释放）
- **镜头**：简洁的"通过。"二字
- **反应**：全人类狂欢，老王抱人转圈，苏小暖嚎啕大哭，瑶瑶眼角泪光
- **林启与艾莉丝**：拥抱，"是大家做到的"

## 场景6：夜枭视角（揭示）
- **镜头**：银色飞船上，夜枭与神秘老师对话
- **关键信息**：97.3%是历史最高分；林启的聪明在于"没有引导"
- **伏笔炸弹**："文明审判只是入场券。真正的考验，现在才开始。"

## 场景7：七道裂缝（结尾悬念）
- **镜头**：林启独自望星空，永恒之核发出警告
- **危机**：太阳系边缘同时出现七个空间裂缝，每个等同于一个完整清算者
- **结尾**：林启紧急召集核心团队，格雷戈尔被提及
- **氛围**：从欢庆骤然转冷，骨寒级悬念

## 情感曲线
```
紧张 → 冷静分析 → 反转（不给答案）→ 感动（信念汇聚）→ 狂喜（通过）→ 骤冷（新危机）
```

## 本章矛盾设计
| 矛盾 | 内容 |
|------|------|
| **核心矛盾** | 开放题 vs 一致性要求 |
| **策略矛盾** | 统一引导 vs 让人说真话 |
| **信息矛盾** | 审判通过的喜悦 vs "只是入场券"的真相 |
| **节奏矛盾** | 欢庆高潮 → 冰冷现实 |
