## 核心设定

### 1. 克苏鲁神明体系

```
【表宇宙】可观测宇宙（人类活动范围）
    ↓
【里宇宙】克苏鲁领域（维度夹缝）
    ↓
【旧日支配者】睡眠中...
    ↓
【外神】正在苏醒...
```

**核心冲突**：旧日支配者即将苏醒，人类只是棋子

## 势力详解

### 人类联邦（主角阵营）

```
首都：地球·联合国广场
军事：联邦舰队（10000艘战舰）
经济：星际贸易
政治：民主选举，勾心斗角

特色兵种：
- 外交官（嘴炮）
- 佣兵（拿钱办事）
- 观察者（卧底）

影射：美国/联合国
```

### 银链帝国（敌对）

```
首都：泰坦星·黄金王座
军事：帝国舰队（50000艘战舰）
政治：帝制，军国主义
皇帝：莱昂萨利（300岁）

特色兵种：
- 铁血卫队（近战）
- 闪电舰队（速度）
- 基因战士（强化人）

影射：德国/苏联
```

### 深空教团（疯狂）

```
圣地：克苏鲁星域（不可名状）
信仰：外神克苏鲁
教宗：不死者

特色：
- 疯狂信徒（自爆）
- 召唤仪式
- 精神控制

影射：中世纪宗教/恐怖组织
```

## 核心爽点设计

### 1. 嘴炮退敌
- 主角靠嘴炮搞定敌人
- 各种骚话+谐音梗
- 让敌人心态崩盘

### 2. 逆风翻盘
- 弱势局面翻盘
- 靠智商不靠武力
- 观众拍案叫绝

### 3. 感情线
- 兄弟情（老王）
- 爱情线（艾莉丝）
- 主仆情（小Q）
- 师生情（张博士）

### 4. 宇宙奇观
- 星际战争场面
- 文明碰撞
- 宇宙恐怖（克苏鲁）

## 写作风格指南

### 轻松幽默写法

```
1. 主角内心吐槽
   "这波啊，这波是..."
   
2. 队友神补刀
   "长官，你的智商像余额一样感人"
   
3. 反差萌
   敌人很帅但很蠢
   
4. 谐音梗
   银链帝国→银链弟国
   深空教团→深空叫团
```

### 煽情写法

```
1. 牺牲前的独白
2. 战友离别
3. 胜利后的空虚
4. 家人等待
```

### 宇宙感写法

```
1. 宏观描写
   "舰队如繁星般划过宇宙"
   
2. 微观细节
   "舷窗外，星光如同永远不会熄灭的尘埃"
   
3. 尺度对比
   "人类引以为傲的舰队，在宇宙眼中不过是沧海一粟"
```
