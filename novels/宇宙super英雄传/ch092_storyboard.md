## 场景二：林启的演讲

| 项目 | 内容 |
|------|------|
| 时长 | 约5分钟 |
| 场景 | 思维空间 |
| 人物 | 林启（主讲）、全体人类 |
| 动作 | 林启悬浮空中，慷慨陈词 |
| 情绪 | 激情、震撼 |
| 音乐 | 渐进式交响乐 |

**分镜2-1**
- 画面：林启深呼吸，准备开口
- 文案：「各位，在选择之前，我想先说一句……」

**分镜2-2**
- 画面：林启开始分析A选项的危害
- 文案：「如果我们选择A，那就意味着我们认同'为了大多数人的利益可以牺牲少数人'这个逻辑……」
- 特效：数字在林启身后浮现：1→10→100→10000

**分镜2-3**
- 画面：林启抛出核心问题
- 文案：「他们问的是——你认为什么是'人'……是三十亿个统计学数字？还是三十亿个活生生的、有名有姓的人？」
- 特效：数字变成人形剪影

**分镜2-4**
- 画面：林启总结
- 文案：「这个题目的关键，不在于A还是B。而在于——我们到底要成为一个什么样的文明。」
- 特效：光芒从林启体内爆发
