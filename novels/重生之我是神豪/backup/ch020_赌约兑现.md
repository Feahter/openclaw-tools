# 第20章 赌约兑现

本章爽点：
1. 一个月到期，投资回报率碾压
2. 秦昊输掉10亿，当众下跪
3. 秦家颜面尽失，走下坡路
4. 宣布收购秦氏集团，爽到极致

---

一个月后，秦昊与周明轩的赌约到期日终于来临。

东海市最顶级的酒店会议厅里，聚集了全市所有的媒体和商界名流。

这场赌约，早就成为了整个东海市茶余饭后的热议话题。

而今天，就是见证结果的时候。

"明轩，准备好了吗？"苏雨晴紧张地握着周明轩的手。

"放心。"周明轩给了她一个安心的眼神，"今天，是秦家父子的末日。"

两人携手走进会场。

会议厅里已经坐满了人，秦建国和秦昊坐在第一排，脸色都不太好看。

这一个月，秦氏集团的股价大跌，市值蒸发了将近100亿。

而这一切，都是因为那场赌约。

"周明轩来了！"不知道谁喊了一声，全场的目光都集中在了门口。

周明轩一身黑色西装，帅气的脸庞上带着淡淡的微笑。苏雨晴一袭白色礼服，美得让人窒息。

两人站在一起，就是全场最靓丽的风景线。

"好了，人都到齐了。"主持人走上台，"现在，让我们公布赌约结果。"

大屏幕上，开始显示两人的投资数据。

首先是秦昊：

【投资总额：10亿】
【最终收益：10.5亿】
【回报率：5%】

看到这个数据，全场一片哗然。

"有没有搞错？5%？这也太少了吧？”

"就是啊，之前不是说他有52%的回报率吗？”

"看来是吹牛逼的，实际操作起来就这水平？”

秦昊的脸色瞬间变得铁青。

这一个月，他确实投资了一些项目，但回报率远远不如预期。

只有5%，已经是最好的结果了。

而秦建国则是叹了口气，儿子的水平，他心里清楚。

"现在，让我们看看周明轩周先生的投资数据！"主持人激动地道。

大屏幕闪烁，数据出现：

【投资总额：10亿】
【最终收益：52亿】
【回报率：420%！】

"420%？！我没看错吧？”

"420%！这还是人吗？！”

"太夸张了！一个月啊，10亿变52亿！”

全场炸锅了！

420%的回报率，碾压秦昊的5%，整整84倍！

"不可能！"秦昊突然跳起来，"这一定是假的！你一定作弊了！”

"作弊？"周明轩冷笑一声，"秦昊，愿赌服输，你该不会是输不起吧？”

"我！"秦昊哑口无言。

"10个亿，拿出来吧。"周明轩淡淡道，"我明轩投资集团正好缺流动资金，多谢秦大少了。”

"你！"秦昊还想说什么，秦建国突然站了起来。

"10个亿，我们秦家出了！"秦建国阴沉着脸，"但是周明轩，你不要太过分！”

"过分？"周明轩笑了，"秦建国，是你的儿子主动要跟我赌的，赌约还是他提出来的。怎么，输了就不认账？”

"你！"秦建国哑口无言。

确实，这场赌约是秦昊主动提出来的，怨不得别人。

"爸，不能给他！"秦昊着急地道，"10个亿啊，我们秦家没有那么多的现金流！”

"闭嘴！"秦建国低喝，"我自有分寸！”

他深吸一口气，看向周明轩："周明轩，10个亿我可以给你。但是我有一个条件。”

"条件？"周明轩似笑非笑，"你觉得你有资格跟我谈条件？”

"你！"秦建国咬紧牙关，"算我求你。”

"求我？"周明轩故作惊讶，"秦建国，你秦氏集团的董事长，竟然求我？”

全场一片寂静。

所有人都没想到，周明轩竟然敢这么跟秦建国说话。

要知道，秦建国在东海市商界可是泰山北斗级别的人物！

而周明轩，一个22岁的大学生，竟然完全不把他放在眼里！

"好，很好。"秦建国气的浑身发抖，"周明轩，你够狠！”

"谢谢夸奖。"周明轩淡淡道，"10个亿快点打过来，我还有事要宣布。”

秦建国深吸一口气，对助理道："给他转账。”

几分钟后，周明轩的账户里多了10个亿。

"好了，现在赌约结束，我还有一件事要宣布。"周明轩走上台，接过话筒。

全场安静下来，所有人的目光都集中在周明轩身上。

"我宣布，"周明轩淡淡道，"明轩投资集团将启动对秦氏集团的收购计划。”

"什么？！”

全场再次炸锅！

收购秦氏集团？！

那可是东海市排名前十的大集团，市值上千亿！

而周明轩竟然要收购它？！

"周明轩，你不要太过分！"秦建国怒吼道。

"过分？"周明轩冷笑，"秦建国，这只是一个开始。你们秦家欠我的，我会一件一件讨回来。”

"你！"秦建国气的吐血，"我跟你拼了！”

他朝着周明轩冲了过去，但被韩子墨一把拦住。

"秦建国，动手可是犯法的。"韩子墨冷笑道，"而且，你觉得你打得过明轩？”

秦建国这才冷静下来，环顾四周，发现周围的保安都虎视眈眈地看着他。

"好，周明轩，你很好！"秦建国深吸一口气，"我们走！”

他拉着秦昊就要离开。

"等等。"周明轩突然道。

秦建国回头："怎么？后悔了？”

"后悔？"周明轩笑了，"我只是提醒你，以后见到我，最好绕着走。”

"你！"秦建国还想说什么，但最终还是忍住了。

带着秦昊，灰溜溜地离开了会场。

而全场响起了雷鸣般的掌声。

"明轩，你太棒了！"韩子墨激动地道。

"周少威武！"周围的人纷纷鼓掌。

苏雨晴走到周明轩身边，眼中满是崇拜："明轩，你太厉害了。”

"这只是开始。"周明轩握住她的手，"雨晴，我会让你成为世界上最幸福的女人。”

苏雨晴幸福地笑了。

【叮！宿主完成打脸成就，奖励：20亿元】

【叮！宿主宣布收购秦氏集团，奖励：50亿元】

系统的提示音不断响起，但周明轩已经完全不在意了。

他的目光看向远方，嘴角露出一抹冷笑。

秦氏集团，这只是开始。

总有一天，整个东海市，都将在他周明轩的掌控之中！

而秦家父子，注定会成为他成功道路上的垫脚石。
