# 第19章 游艇岛屿

本章爽点：
1. 100亿买下私人岛屿，极致奢华
2. 邀请好友度假，圈子高级感拉满
3. 秦昊想加入被直接拒绝
4. 浪漫表白，苏雨晴感动

---

"明轩，你真的买了一座岛？！"韩子墨接到周明轩的电话，直接蹦了起来。

"没错。"周明轩笑道，"马来群岛那边的一个私人岛屿，风景优美，我准备把它改造成度假村。怎么样，有没有兴趣一起去玩玩？”

"必须有兴趣啊！"韩子墨激动地道，"我马上请假！”

"好，我让助理给你订机票。”周明轩道，"把陈晓晓也带上。”

"没问题！"韩子墨嘿嘿一笑，"明轩懂我！”

挂掉电话，周明轩看向苏雨晴："雨晴，这次只有我们几个自己人，没有外人。”

苏雨晴心中一暖："明轩，谢谢你。”

"谢什么。”周明轩握住她的手，"你是我的女人，我想把最好的都给你。”

苏雨晴俏脸微红，但眼中满是幸福。

三天后，一架私人飞机降落在马来群岛的一个小型机场。

周明轩包下了一艘豪华游艇，带着苏雨晴、韩子墨和陈晓晓，一行人朝着那座私人岛屿驶去。

"明轩，这岛多少钱？”韩子墨好奇地问。

"100亿。”周明轩淡淡道。

"100亿？！”韩子墨差点没跳起来，"我靠，你也太夸张了吧！”

"还好。”周明轩道，"这座岛风景很好，我准备开发成会员制度假村，只对顶级富豪开放。”

"牛！”韩子墨竖起大拇指，"这才是神豪的格局！”

游艇靠近岛屿，众人只见岛上绿树成荫，沙滩细腻，海水湛蓝，美得像画一样。

岛屿上已经建好了一座豪华别墅，还有私人码头、停机坪、网球场、游泳池等设施。

"我的天！”陈晓晓都震惊了，"这也太豪华了吧！”

"这还算一般。”周明轩道，"我打算再投资50个亿，把这里改造成全球最顶级的度假胜地。”

苏雨晴看着这一切，眼中满是震撼。

她虽然是苏氏集团的千金，但也没有这么奢侈过。

"明轩，你对我太好了。”她忍不住道。

"傻瓜。”周明轩刮了刮她的鼻子，"你是我的女人，不对你好对谁好？”

苏雨晴幸福地笑了。

接下来几天，众人尽情享受着岛屿的阳光、沙滩和海水。

白天，他们潜水、钓鱼、冲浪。

晚上，他们举办烧烤派对，在星空下喝酒聊天。

这天傍晚，周明轩带着苏雨晴来到了岛屿最高点。

这里可以看到整个岛屿的美景，夕阳把海面染成了金色，美得让人窒息。

"雨晴。”周明轩突然道。

"嗯？"苏雨晴转头看他。

周明轩深吸一口气，从口袋里掏出一个盒子。

打开盒子，里面是一枚巨大的钻戒，在夕阳下闪闪发光。

"雨晴，嫁给我吧。”

苏雨晴愣住了。

她想过周明轩会给她惊喜，但没想到会是求婚！

"明轩......我......"她眼眶红了。

"我知道这很突然。”周明轩道，"但我想告诉你，我是认真的。我想一辈子对你好，宠你，疼你，让你成为世界上最幸福的女人。”

"我愿意！”苏雨晴泪流满面，"我愿意！”

周明轩把戒指戴在她手上，然后紧紧拥抱着她。

夕阳下，两人相拥的画面美得就像一幅画。

【叮！宿主求婚成功，奖励：10亿元】

系统的提示音响起，但周明轩已经完全不在意了。

此刻在他的世界里，只有苏雨晴。

"明轩，我爱你。”苏雨晴轻声呢喃。

"我也爱你。”周明轩吻住了她的唇。

就在这浪漫的时刻，一个不和谐的声音突然响起。

"周明轩！”

众人回头，只见秦昊不知道什么时候出现在了岛上，身后还跟着几个保镖。

"你怎么在这里？！”韩子墨怒道。

"我为什么不能在这里？”秦昊冷笑道，"这座岛又不只有你们能来。”

"这座岛是我的。”周明轩淡淡道，"我不记得邀请过你。”

"你！”秦昊表情一僵，"周明轩，你不要太嚣张！我今天来是给你面子的！”

"面子？”周明轩像看傻子一样看着他，"秦昊，你搞清楚状况。这是我的私人岛屿，未经允许擅自闯入，我可以报警抓你。”

"你！”秦昊咬紧牙关，"周明轩，你不要太得意！我今天来是想告诉你，秦氏集团不会放过你的！”

"哦？”周明轩笑了，"那又怎样？”

"你！”秦昊被他这种无所谓的态度气得半死，"好，你等着！”

他转身就要走。

"等等。”周明轩突然道。

秦昊以为他怕了，得意地道："怎么？知道怕了？”

"谁给你的勇气未经允许上我的岛？”周明轩冷声道，"子墨，叫保安把他们轰出去。”

"好嘞！”韩子墨早就看不惯秦昊了，立刻叫来岛的保安。

"是，先生！”

几个彪形大汉冲上来，不由分说地把秦昊和他的保镖架了起来。

"周明轩，你干什么？！”秦昊挣扎着，"放开我！”

"扔出去。”周明轩淡淡道。

"是！”

秦昊被像扔垃圾一样扔出了岛屿。

"周明轩，我跟你没完！”远处传来他的怒吼声。

周明轩懒得理会，这种跳梁小丑，根本不值得他放在心上。

"明轩，会不会有什么事？”苏雨晴有些担心。

"放心。”周明轩握住她的手，"有我在，没人能伤害你。”

苏雨晴点点头，把头靠在他肩膀上。

而此时，秦昊正站在岛屿外的海面上，一艘小船里，脸色阴沉得可怕。

"周明轩今日之辱，我秦昊记下了！”
