# 分镜脚本模板 v4.0（百万字版+七宗罪）

---

## 📌 模板说明

> 本模板适用于《重生之我是神豪》v4百万字版本

---

## 📝 每章必做清单

```
□ 1. 场景连续性
□ 2. 时间连续性  
□ 3. 情绪连续性
□ 4. 因果连续性
□ 5. 信息连续性
□ 6. 爽点设计（3-4个）
□ 7. 七宗罪对应
□ 8. 情感场景（每章1个）
□ 9. 字数3500-4000字
```

---

## 📊 章节模板（3500字）

```
【章节】chXXX
【类型】日常章/情感章/高潮章

【起】(800字)
- 场景：
- 人物：
- 目标：

【承】(1000字)
- 阻碍：
- 行动：
- 情感线：

【转】(1200字)
- 爽点1（七宗罪）：
- 爽点2：
- 爽点3：
- 打脸：

【合】(500字)
- 解决：
- 情感升华：
- 结尾悬念：
```

## 🎭 情感场景模板

### 模板1：邂逅

```
【场景：相遇】

环境：[咖啡厅/雨天/车内]
氛围：[心动/暖味]

她的出现：
- [穿着]
- [外貌]
- [气质]

目光交汇：
- 四目相对的瞬间
- 心理活动
- 身体反应

【转场】
```

### 模板2：亲密

```
【场景：独处】

环境：[家里/车内/海边]
氛围：[暖味/心动]

身体接触（点到为止）：
- 牵手
- 拥抱
- kiss

心理描写：
- 主角感受
- 她的反应

【转场：下一章】
```
