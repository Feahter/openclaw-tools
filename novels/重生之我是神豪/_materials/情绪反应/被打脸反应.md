```

脖子缓缓转过来。

像一部卡顿的纪录片。

三秒钟。

什么都没说。

嘴巴张了张。

又闭上了。
```

**重复确认**
```

"这辆车。"

"劳斯莱斯幻影。"

"您确定？"

"全款。"

这两个字。
```
