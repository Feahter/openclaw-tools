# 消费流程素材

## 验资场景

### 银行验资

**被刁难**
```
"您好，我想取钱。"

"多少？"

"500万。"

柜员抬起头。

看了我一眼。

"提前预约了吗？"

"没有。"

"那不行。"

"500万以上需要提前一天预约。"

她公事公办地说。

"那怎么办？"

"您明天再来吧。"

语气很敷衍。

好像在说：穷逼，取什么500万。

我没说话。

掏出手机。

打了个电话。

五分钟后。

支行行长亲自出来了。

"周先生！"

"您要取钱？"

"不用取了。"

我说。

"帮我把这500万换成现金。"

"我要数着玩。"
```

**行长亲自**
```
"周先生，这边请！"

行长亲自带我进VIP室。

"小王，给周先生倒水！"

"好的！"

刚才那个柜员。

现在像个小透明。

"周先生，您稍等！"

"500万马上到！"

五分钟后。

桌面上摆满了现金。

500万。

红的、绿的、紫的。

"您点点？"

"不用了。"

我随手抓了一把。

"赏你了。"

行长笑得眼睛都没了。

"谢谢周先生！"
```

### 房产验资

**看房被讽**
```
"这套房子什么价？"

我指着墙上最贵的那套。

"哦，那套啊。"

销售过来了。

"3个亿。"

"3个亿？"

"嗯，全款。"

她重点强调了"全款"两个字。

好像在提醒我：

你买不起。

"可以看看吗？"

"可以啊。"

她带我上去。

全程讲解。

很敷衍。

走到一半。

"算了，不看了。"

我说。

"太便宜了。"

"啊？"

销售愣住。

"这片有更贵的吗？"
```

### 私人会所验资

**会所门槛**
```
"先生，请出示会员卡。"

门口保安拦住我。

"没有。"

"那不能进。"

"我是来找人的。"

"找谁？"

"周明轩。"

"周...周先生？"

保安愣了一下。

"请稍等。"

他打电话确认。

然后态度变了。

"周先生，请！"

原来。

这会所。

就是我开的。
```

## 拍卖会场景

### 拍卖会类型

| 类型 | 门槛 | 常见物品 |
|------|------|---------|
| 艺术品拍卖 | 1000万资产证明 | 字画、古董 |
| 珠宝拍卖 | 500万资产证明 | 钻石、翡翠 |
| 房产拍卖 | 200万保证金 | 豪宅、法拍房 |
| 跑车拍卖 | 50万保证金 | 限量超跑 |

### 拍卖流程

**入场**
```
"先生，请出示邀请函。"

门口礼仪小姐微笑道。

"没有。"

"那...您有资产证明吗？"

"什么资产证明？"

"比如...银行卡流水。"

我掏出手机。

打开银行APP。

余额：10,000,000,000+

1个亿。

"这个够吗？"

礼仪小姐的笑容僵住了。

"够...够！"

"您请进！"

原来。

1个亿。

可以买下整个拍卖会。
```

**叫价**
```
"500万！"

"600万！"

"800万！"

...

这幅画。

起拍价100万。

现在800万了。

"1000万。"

我举牌。

全场安静。

"1000万第一次！"

"1000万第二次！"

"1000万第三次！"

"成交！"

"恭喜这位先生！"

成交价：1000万。

画的是什么东西。

我都没看清。

但是。

爽。

千金难买我高兴。
```

### 拍卖爽点

**全场焦点**
```
"1000万！"

"又是这位先生！"

"他今晚已经拍了第三件了！"

"前面两件：3000万！"

"这幅画：1000万！"

"总消费：4000万！"

周围的人议论纷纷。

"这人谁啊？"

"不知道，太有钱了！"

"一晚上花4000万？"

"这就是顶级富豪吗？"

我坐在座位上。

表情平静。

内心爽的一批。

4000万。

对。

就是我。

今晚的消费。
```

## 高端场所消费

### 顶级酒店

**总统套房**
```
"总统套房多少钱一晚？"

"8万。"

"住一个月呢？"

"240万。"

"行。"

我刷卡。

"住一年。"

240万 × 12 = 2880万。

"好的！"

前台手续办得飞快。

"这是您的房卡！"

"欢迎入住！"

2880万。

住一年。

眼睛都没眨。

服务员的眼神。

像在看神。
```

### 高端餐厅

**米其林三星**
```
"先生几位？"

"一位。"

"好的，这边请。"

被带到一个小角落。

"那个位置有人订了。"

解释得很敷衍。

行吧。

我将就坐。

吃完饭。

"买单。"

"您好，一共3万8。"

"刷卡。"

滴——

"支付成功。"

"谢谢光临！"

我站起身。

走出去。

门口那桌。

刚来。

法拉利钥匙拍在桌上。

服务员笑脸相迎。

"先生几位？"

"三位。"

"里面请！"

我看在眼里。

没说话。

三年后。

老子也让你这么对我。
```
