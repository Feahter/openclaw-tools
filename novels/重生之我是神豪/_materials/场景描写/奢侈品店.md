- 门脸：黑底金字logo，橱窗展示当季新品
- 门口：专人开门，佩戴白手套
- 地面：深色木地板或理石

### 销售员反应

**反应1：冷眼**
```
我推门进去。

"欢迎光临。"

语气很敷衍。

一个sales正在整理货架，
扫了我一眼，继续忙自己的。

也是。
我穿的是优衣库99元T恤，
裤子是淘宝100块的，
鞋子是穿了两年的回力。

整个店里的人都在忙，
就是没人理我。
```

**反应2：轻视**
```
"您想买什么？"

一个sales终于走过来了。

语气像在问"你买得起吗"。

"看看。"我说。

"哦。"

她应了一声，站在三米外，
抱着臂，没有要介绍的意思。

好像我随时会偷东西似的。
```

**反应3：震惊**
```
"买单。"

我把选好的东西放在柜台上。

Neverfull、Speedy、钱包、丝巾...
总共12万。

那个sales的动作僵住了。

"您...您确定？"

"刷卡。"
```

```

"30万够吗？"
```

## 场景通用模板

### 进店前
```
门口。

我看了一眼手机导航。

"就这？"

LV的logo在阳光下闪闪发光。

深黑色的门头，
金色的Louis Vuitton几个大字，
门口站着一个戴白手套的门童。

我低头看了看自己。

优衣库T恤，九分裤，
一双穿了两年的运动鞋。

确实有点不搭。

但那又怎样？

老子有钱。
```

### 被轻视时
```
"欢迎光临。"

标准的笑容，标准的语气。

但眼神不对。

她盯着我的鞋。

三秒钟。

然后笑容就淡了。

"您随便看。"

说完转身去整理货架了。

好像我会偷东西似的。
```

### 选货时
```
"这个，还有这个不要，其他的包起来。"

我随手点了七八个。

sales的眼睛亮了。

"先生，您确定要这些？"

"有问题？"

"没有没有！我这就帮您包起来！"

她的态度180度大转弯。
```

### 刷卡时
```
滴——

POS机的声音。

"支付成功——"

我看到sales的手指在发抖。

12万。

对她们来说可能是半年工资。

对我而言：
呼吸而已。
```
