## 私人飞机体验

### 登机流程

**安检**
```
普通民航安检：
排队、拖鞋、脱衣服、搜身...

私人飞机安检：

"周先生，这边请。"

专属通道，不用排队。

一个漂亮空姐带我走进去。

"您先休息一下，飞机准备好了我叫您。"

她把我领进VIP候机室。

独立空间，真皮沙发。

咖啡、红酒、水果...

这才是人过的日子。
```

**登机**
```
登机口停着一架飞机。

湾流G550。

机身白的发亮。

"周先生，请。"

空姐伸出手，扶我上舷梯。

机舱门打开。

我踏进去。

艹。

这哪是飞机。

这是空中宫殿。
```

## 飞行体验

### 起飞
```
"周先生，我们准备起飞了。"

广播响起。

舷窗外的跑道开始后退。

速度越来越快。

突然。

失重感传来。

飞机离地了。

我升空了。

从舷窗看下去。

地面上的房子像蚂蚁。

车像玩具。

这就是权力的感觉。

在天上。

老子天下第一。
```

### 云端体验
```
平飞之后。

我站起身。

在机舱里走动。

一共有12个座位。

真皮+木纹+金属。

"周先生，您需要休息吗？"

"嗯。"

我走进卧室。

1.8米大床。

床单是埃及棉。

400针。

躺下去。

软。

太软了。

比五星级酒店的床还舒服。

空姐走进来。

"周先生，温度可以吗？"

"可以。"

"好的，有需要叫我。"

然后轻轻带上门。

这就是包机的爽点：

全程一对一服务。

Call一下就到。

比男朋友还贴心。
```

## 对比描写

### vs 普通航班
```
上次坐经济舱。

提前2小时到。

排队值机。

排队安检。

排队登机。

找座位。

放行李。

坐下。

座位窄。

腿伸不直。

旁边还有人打呼噜。

空调吹得头疼。

3小时飞行。

像坐了3年牢。

/

这次坐私人飞机。

提前20分钟到。

专车送到舷梯下。

登机。

起飞。

3小时后。

到了。

全程VIP服务。

香槟红酒。

躺椅按摩。

这就是差距。
```

```

"你...你坐私人飞机？"

"不可能！"

"这架飞机...是你的？"

"租的。"我说，"一天80万。"

80万。

他一年的生活费。

对我而言：

就是一小时的事。
```
