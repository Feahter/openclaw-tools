## 游艇派对场景

### 登船场景

**被拦下**
```
"先生，请留步。"

码头门口，两个保安拦住我。

"这是私人游艇会。"

"我知道。"我说，"我朋友在这。"

"请问您朋友是？"

"周明轩。"

"周...周先生？"

保安愣了一下。

"您稍等，我确认一下。"

他拿起对讲机说了几句。

然后态度180度大转弯。

"周先生，请！游艇已经准备好了！"

原来，
这艘游艇就是我的。
```

**登船**
```
我踏上甲板。

柚木地板，光可鉴人。

"欢迎登船！"

十多个服务员鞠躬。

一律白色制服，
男的高大帅气，
女的身材苗条。

这是游艇派对的标准配置。

"周先生，这边请。"

管家带我走进船舱。
```

### 船舱设施

**豪华程度**
```
船舱里是另一番景象。

真皮沙发，实木家具。

KTV包厢、酒吧台、电影院...

甚至还有一间麻将房。

"周先生，您想先喝点什么？"

"随便。"

"好的，拉菲可以吗？"

"可以。"

82年的拉菲，
这艘游艇上有一整柜。

价值：
300万。
```

**泳池**
```
二层甲板有个泳池。

不是那种小泳池。

是25米的无边泳池！

水是蓝色的，
在阳光下闪闪发光。

"周先生，水温帮您调到28度了。"

"嗯。"

我躺在躺椅上。

看着蓝天白云。

这就是顶级富豪的生活。

老子终于体验到了。
```

### 派对场景

**热闹场面**
```
游艇上已经来了几十个人。

网红、模特、富二代...

清一色的俊男美女。

"轩哥！"

韩子墨跑过来。

"你终于来了！"

"怎么样？"

"牛逼！"

他竖起大拇指。

"这艘游艇，租一天80万！"

80万。

我眉头都没皱一下。

"喜欢吗？"

"喜欢！"

"送你了。"
```

**泳装派对**
```
三层甲板是日光浴区。

一群比基尼美女在晒太阳。

大长腿，马甲线，A4腰。

看得人眼花缭乱。

"周哥！"

一个网红朝我招手。

"来这边坐！"

我走过去。

一堆美女给我让位置。

这种众星捧月的感觉...

爽。

太爽了。
```

## 被打脸场景

### 嘲讽
```
"这游艇是你租的吧？"

秦昊不知道什么时候上了船。

"租一天多少钱？5万？10万？"

他冷笑。

"打肿脸充胖子。"

我没理他。

"保安。"

我打了个响指。

"把这个人给我请下去。"

"凭什么？这是公共游艇！"

"这艘船是我的。"

我淡淡地说。

"不是租的。"

"不可能！"

"不信？"

我掏出手机，给他看了游艇购买记录。

1.2亿。

全款。

他脸都绿了。
```
