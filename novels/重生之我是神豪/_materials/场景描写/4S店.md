## 劳斯莱斯展厅

### 展厅环境
- 2000-5000平米独立展厅，层高8-12米
- 水晶吊灯/艺术吊灯，意大利进口
- 地面：大理石/拋光混凝土
- 灯光：spotlight聚焦每辆车
- 背景音乐：古典乐/轻爵士，音量控制在30分贝
- 温度：22-24度，恒温

### 销售员反应模板

**反应阶段1：轻视**
```
"先生，有预约吗？"

她抱臂站着，居高临下地看着我。
优衣库T恤，洗得发白。
安踏运动鞋，鞋边还有灰。

"没有就不能进？"我问。

"能进，当然能进。"

嘴上这么说，脚却没动。
眼神从头扫到脚，又从脚扫回头。
那表情就像在说：又一个来拍照的穷逼。
```

**反应阶段2：愣住**
```
"这辆，曜影，帮我开门。"

销售的脚像被钉住了。

她维持着抱臂的姿势，脖子缓缓转过来，
像一部卡顿的纪录片。

"您...说什么？"

"曜影。"我重复，"敞篷那辆。"
```

**反应阶段3：震惊**
```
滴——

POS机的声音在寂静的展厅里像一道惊雷。

"支付成功——"

我看到她的手指在发抖。

那双手曾经多么优雅地抱臂，
现在像触电一样颤抖着递过来pos单。
```

**反应阶段4：谄媚**
```
"先生，您稍等，我给您倒杯水！"

她小跑着去倒水，12厘米的高跟鞋
在理石地面上敲出急促的节奏。

"您坐这休息一下，我帮您办手续！"

180度大转弯。
刚才的冷脸变成了笑脸，
眼睛笑成月牙，
嘴角快咧到耳朵根了。
```

## 细节素材

### 销售员装备
- 工牌：姓名、职位、照片
- 西装：阿玛尼、杰尼亚定制
- 袖扣：梵克雅宝、卡地亚
- 手表：浪琴、欧米茄
- 皮鞋：约翰罗布、Berluti

### 接待流程
1. 迎宾：3米内主动迎接
2. 询问：先生/女士贵姓
3. 落座：VIP区
4. 饮品：依云水/巴黎气泡水/现磨咖啡
5. 介绍：绕车讲解
6. 试驾：专人驾驶
7. 报价：计算器/平板展示
8. 签约：VIP室

### 支付方式
- 刷卡：pos机打印凭条"滋——"一声
- 转账：手机银行到账提示
- 现金：保安护送去银行
- 贷款：销售脸色骤变
