## 第一章：太平洋上空

商务机的舷窗外，是一望无际的太平洋。

陈默靠在座椅上，手指轻轻敲击着扶手，脑海中飞速运转。

"微软CEO亲自邀请，这排场不小啊。"他喃喃自语。

系统面板自动弹出：

```
【微软总部情报分析】
纳德拉接机概率：97.3%
欢迎仪式规模：预计50人以上媒体
暗流概率：82.7%
建议：保持警惕，观察接机人群中的可疑人物
```

陈默挑了挑眉。

"82.7%的暗流概率？这不是明摆着是鸿门宴吗？"

坐在对面的是周伟，此刻正抱着一台笔记本疯狂敲代码。

"默哥，我查过了，"周伟压低声音，"微软总部最近戒严了三次，安保级别提升了一级。而且......"

"而且什么？"

"索罗斯基金的代表，秘密进入微软总部两次。"

陈默笑了。

"有意思。道歉是假，试探是真。这位纳德拉先生，是想摸摸我的底啊。"

"那我们还去？"周伟皱眉。

"当然要去。"陈默端起咖啡杯，"我不去，怎么知道他们想玩什么？"

他望向窗外。

云层之下，是旧金山的轮廓。

这座城市，曾经是全球科技的中心。现在，也该换个主人了。

## 第二章：接机暗流

旧金山国际机场，阳光明媚得有些刺眼。

陈默走下舷梯的那一刻，立刻感受到了"热情"的气氛。

红毯两侧，数十家媒体的闪光灯此起彼伏。人群中，他敏锐地捕捉到了几个不寻常的身影——

西装革履却站姿笔挺的，明显是安保人员；假装记者却不停打量他的，是在评估他的威胁等级；还有几个......

"职业杀手。"系统在视网膜上标注，"建议保持3米以上距离。"

陈默面色如常，稳步走下红毯。

"陈先生！欢迎来到旧金山！"

一个温和的声音传来。

萨提亚·纳德拉，微软CEO，此刻正微笑着张开双臂。

五十多岁的中年人，头发花白，眼神深邃。那种城府极深的感觉，让陈默想起了另一句话——

**真正的猎人，往往以猎物的方式出现。**

"纳德拉先生，久仰久仰。"陈默握手，保持微笑，"您亲自来接机，这让我很意外啊。"

"应该的。"纳德拉笑着说，"陈先生是贵客，我们微软有失远迎的地方，还请多多包涵。"

两人并肩走向车队。

就在此时，陈默的余光捕捉到了一个身影——

人群中，一个戴着棒球帽的男人，低着头，却始终保持着与他的视线平行。

**可疑人物：评估威胁等级中......**

系统面板闪烁：

```
【身份识别】
姓名：迈克尔·罗宾逊（推测）
身份：量子对冲基金创始人、索罗斯嫡传弟子
威胁等级：极高
状态：伪装接近中
```

陈默心中冷笑。

好啊，老朋友来了。

"纳德拉先生，"他忽然停下脚步，"贵国的安保措施，真是让人印象深刻啊。"

纳德拉的笑容僵了一瞬。

"陈先生什么意思？"

"没什么。"陈默耸耸肩，"就是觉得，记者群里混进来几只苍蝇，有点煞风景。"

远处，那个戴棒球帽的身影，转身消失在人群中。

纳德拉的眼神变了变，随即恢复如常。

"陈先生果然敏锐。"他做了个请的手势，"车上聊？"

"好啊。"陈默笑了。

**这场游戏，越来越有意思了。**

## 第三章：微软总部

微软雷德蒙德总部，会议室。

陈默坐在长桌的一侧，周伟站在他身后。对面是纳德拉，以及几位微软高管。

"陈先生，"纳德拉推开一份文件，"这是我们微软的诚意。"

文件标题：《Aether战略合作框架》

陈默扫了一眼，嘴角勾起一抹弧度。

**【文件分析】**

```
核心条款分析：
1. 联合开发Aether：微软获得50%代码控制权 → 实际是夺取控制权
2. 市场划分：中国市场归陈默，全球其他市场微软主导 → 分化策略
3. 专利共享：单向开放，微软可使用Aether专利 → 不对等条约
4. 董事会席位：微软要求2席，陈默团队1席 → 夺权意图明显

综合评估：披着合作外衣的收购协议
```

"诚意？"陈默放下文件，"纳德拉先生，这诚意我有点看不懂啊。"

"陈先生有何高见？"

陈默站起身，走到窗边。

窗外是西雅图标志性的雨景，灰蒙蒙的，像极了某些人的心思。

"贵方的条款，总结起来就是一句话——"他转身，目光锐利，"微软想要Aether的控制权，又怕直接抢会引起众怒。所以搞了个'合作'的名头，让我主动交出控制权。"

会议室里一片寂静。

纳德拉的笑容终于消失了。

"陈先生果然聪明。"他靠向椅背，"但聪明人应该知道，有时候，识时务者为俊杰。"

"哦？"陈默挑眉，"纳德拉先生这是在威胁我？"

"不敢。"纳德拉摇摇头，"只是提醒陈先生，微软在AI领域的底蕴，不是几个月就能积累的。你有Aether，但我们有Azure，有Office，有全球最大的开发者生态。"

他顿了顿。

"合作，是双赢。对抗......"

"对抗会怎样？"陈默打断他。

"微软会联合所有能联合的势力，在每一个市场狙击Aether。"纳德拉的眼神变得锐利，"陈先生应该不想看到那一天吧？"

陈默沉默了。

空气中弥漫着火药味。

三秒后——

他笑了。

"纳德拉先生，您说的很有道理。"陈默重新坐下，"但我有个问题。"

"请说。"

"您刚才说的那些，是微软董事会的意思，还是......您个人的意思？"

纳德拉皱眉。

"有什么区别？"

"区别大了。"陈默掏出手机，"如果是董事会的意思，那我可以直接和董事会谈。如果是您的意思......"

他按下播放键。

一段录音响起——

**"量子对冲基金准备就绪，150亿做空资金待命。只要Aether股价下跌，我们立刻进场。"**

**"索罗斯先生那边已经打好招呼，他会配合我们的行动。"**

**"陈默？他以为自己赢了。实际上，游戏才刚刚开始。"**

录音结束。

纳德拉的脸色彻底变了。

"这......这是伪造的！"

"伪造？"陈默收起手机，"纳德拉先生，这段录音是贵公司副总裁史密斯先生的通话记录，我这里还有完整的通讯日志。"

他站起身，居高临下地看着纳德拉。

"微软想和我玩？可以。但前提是——公平游戏。"

"你们在北冰洋对我做的事，在日本做的事，在全球做的事......我都记得。"

"现在，你们想用'合作'的名义来抢我的东西？"

陈默笑了，笑容里带着一丝冷意。

**"抱歉，我陈默，从来不吃这一套。"**

## 第四章：记者会惊变

下午三点，微软总部新闻发布会。

全球数百家媒体齐聚一堂，等待着这场"和解盛会"的召开。

陈默站在后台，透过幕布看着下面的记者群。

"默哥，"周伟凑过来，"苏晚晴姐发来消息，说检测到微软服务器有异常数据流。"

"哦？"

"他们在试图远程访问Aether的核心代码库。"

陈默挑眉。

"这帮人，还真是迫不及待啊。"

他掏出手机，发了一条消息给苏晚晴：

**"让他们访问。访问的同时，植入追踪程序。我要知道他们的服务器在哪里。"**

三秒后，苏晚晴回复：

**"明白。已开始部署。"**

陈默笑了。

**想偷我的东西？那就要做好被反偷的准备。**

新闻发布会开始。

纳德拉走上台，开始了"诚意十足"的道歉。

"过去，微软在一些决策上存在偏差。我们低估了Aether的潜力，也误解了陈默先生的愿景。今天，我代表微软，向陈默先生，向全球AI社区，致以诚挚的歉意......"

陈默站在台下，面带微笑地鼓着掌。

**演技不错。可惜，骗不了我。**

道歉结束，进入记者提问环节。

"陈默先生，"一个记者站起来，"您接受微软的道歉吗？您认为两家公司的合作前景如何？"

陈默接过麦克风。

"当然接受。"他笑着说，"微软是伟大的公司，纳德拉先生也是优秀的管理者。合作嘛......"

他顿了顿，目光扫过台下。

"当然是共赢最好。但如果有人想玩阴的......"

"那我也不会客气。"

台下响起一片窃窃私语。

就在这时——

发布会的大门被推开。

一个身影逆光走了进来。

陈默的笑容僵住了。

那个人......

不，不可能。

那个人，早就死了。

"好久不见，陈默。"

熟悉的声音，熟悉的笑容。

"还记得我吗？"

陈默的瞳孔骤然收缩。

**陆奇。**

不，不是陆奇。

是——

"李明辉。"陈默一字一顿地说出这个名字，"你还没死？"

那个身影笑了。

"死？"他摇摇头，"陈默，你太天真了。"

"天网计划，只是开始。"

"而今天......"

他抬起手，手中是一个熟悉的U盘。

"普罗米修斯计划的核心代码，就在这里。"

全场哗然。

纳德拉的脸色变得惨白。

陈默的眼神，却前所未有的平静。

"好啊。"他说。

"正愁找不到你。"

"既然你自己送上门来了......"

他向前迈出一步。

**"那就别走了。"**
