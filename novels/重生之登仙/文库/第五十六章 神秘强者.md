# 第五十六章 神秘强者

## 分镜脚本

### 场景设计

| 项目 | 内容 |
|------|------|
| **核心画面** | 神秘老者现身，李玄陷入绝境，洛璃倒戈相向 |
| **情绪值** | +4→+3→+5（紧张→危机→反转） |
| **爽点类型** | 绝境逆袭+身份揭秘 |

### 分镜设计

**场景1：神秘老者现身**
- 场景：万族战场深处，空间扭曲
- 人物：神秘老者（合体期以上），李玄，洛璃
- 对话：老者道出混沌灵根的来历
- 氛围：压抑恐惧

**场景2：老者目的**
- 信息1：老者是上古时期的存活者
- 信息2：被困万族战场无尽岁月
- 信息3：想要李玄的混沌灵根重塑肉身
- 冲突：老者对李玄出手

**场景3：绝境求生**
- 危机：李玄身受重伤
- 反转：体内混沌之气自动护主
- 异变：混沌灵根与老者产生共鸣

**场景4：身份反转**
- 震惊：老者认出混沌灵根的真正来历
- 转折：老者态度大变
- 伏笔：老者知晓李玄父亲的下落

### 五项自检

| 检验项 | 内容 | 状态 |
|--------|------|------|
| **场景具体** | 万族战场深处，古老遗迹空间 | ✅ |
| **画面感** | 空间扭曲，黑雾弥漫，老者身影若隐若现 | ✅ |
| **数字感知** | 合体期以上vs炼虚初期，压倒性差距 | ✅ |
| **反派存在** | 神秘老者意图夺取灵根 | ✅ |
| **情感共鸣** | 李玄的绝望与不屈，洛璃的复杂心境 | ✅ |

### 矛盾设计

- **核心矛盾**：李玄vs神秘老者
- **洛璃定位**：墙头草，可能倒戈
- **冲突升级**：实力差距悬殊，李玄命悬一线

“滚出来！”

李玄怒喝。

“年纪轻轻。”

“脾气倒是不小。”

苍老的声音。

在空间中回荡。

“嗖！”

凝聚成。

一个佝偻的身影。

“您...”

“你？”

“也配与本座说话？”

老者冷笑。

“滚一边去。”

“什么是魔。”

发出脆响。

“噗！”

“真是。”

“笑话。”

他在心中呐喊。

堕神一步步走近。

“放弃抵抗吧。”

“归本座。”

“可以让你。”

李玄眼神坚决。

“混沌之气。”

体内混沌之气。

疯狂运转。

堕神轻咦一声。

“有意思。”

“就让我。”

“看看。”

“你的极限。”

“轰！”

李玄气息暴涨。

堕神笑了。

镇压而下。
