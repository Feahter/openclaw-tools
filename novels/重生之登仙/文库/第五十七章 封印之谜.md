# 第五十七章 封印之谜

## 分镜脚本

### 场景设计

| 项目 | 内容 |
|------|------|
| **核心画面** | 堕神传授封印解除之法，李玄得知父亲被困于天道牢笼 |
| **情绪值** | +3→+4→+5（紧张→震惊→决绝） |
| **爽点类型** | 获得重要情报+实力提升 |

### 分镜设计

**场景1：父亲的真相**
- 堕神道出李破天的真正处境
- 天道并非无敌，李破天曾与天道大战
- 父亲被困在"天道牢笼"而非死亡
- 信息：想要救父亲，需要突破仙境

**场景2：封印解除之法**
- 混沌灵根的正确用法
- 需要在万族战场核心布置引灵大阵
- 收集五块上古残碑
- 风险：可能唤醒其他被封印的存在

**场景3：危机降临**
- 封印松动的波动惊动其他存在
- 沉睡的远古魔物苏醒
- 李玄面临腹背受敌的局面

**场景4：绝境突破**
- 在生死危机中突破炼虚后期
- 与堕神联手对抗魔物
- 章尾：更多强者苏醒，虎视眈眈

### 五项自检

| 检验项 | 内容 | 状态 |
|--------|------|------|
| **场景具体** | 万族战场核心，上古封印祭坛 | ✅ |
| **画面感** | 黑雾翻滚，魔影重重，天地变色 | ✅ |
| **数字感知** | 炼虚后期vs合体期魔物，越级挑战 | ✅ |
| **反派存在** | 远古魔物苏醒，目标李玄 | ✅ |
| **情感共鸣** | 父亲未死的激动，拯救父亲的责任感 | ✅ |

### 矛盾设计

- **核心矛盾**：李玄vs远古魔物
- **内部矛盾**：救父心切vs实力不足
- **冲突升级**：一边要解除封印，一边要应对魔物
