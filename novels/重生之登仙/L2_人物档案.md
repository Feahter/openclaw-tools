## 主角

### 李玄（李道玄）

| 属性 | 内容 |
|------|------|
| **身份** | 金光门外门弟子→亲传弟子→五行道祖 |
| **年龄** | 重生时17岁，前世金丹期 |
| **灵根** | 五行俱全灵根（万中无一） |
| **修为** | 练气期（重生后）→金丹期→元婴期→化神期 |
| **主修功法** | 《五行轮转诀》《玄阴心经》《玄阴锻体术》 |
| **特殊能力** | 九龙戏珠之术、道化符、太初道印、寒龙之力 |

**性格特质**:
- 外在：坚韧不拔、谦逊有礼、幽默风趣
- 内在：有责任担当、追求大道、正义感强
- 决策：冷静理性，善于审时度势

**核心欲望**:
- 表层：查明前世被背叛的真相
- 中层：保护身边的人
- 深层：追求真正的"道"，突破命运限制

**人物弧光**:
- 重生前期：隐忍发育、低调行事
- 中期：展现实力、引起关注
- 后期：揭开阴谋、对抗命运
- 最终：超越自我、踏上真正的修仙大道

**标志性台词**:
- "困境即是机遇，让我们换个角度思考。"
- "天外有天，人外有人，唯有虚心求教方能长进。"
- "百败而不馁，千折而不挠。"

## 反派

### 玄冥真人（最大反派）

| 属性 | 内容 |
|------|------|
| **身份** | 金光门太上长老、主角师尊（前世） |
| **修为** | 化神期（后期隐藏实力） |
| **性格** | 阴险狡诈、城府极深、追求极致力量 |
| **与主角关系** | 师尊（前世）→敌人 |

**人物特点**:
- 前世背叛主角，夺取金丹
- 背后有惊天阴谋
- 真实身份是玄冥魔宗余孽
- 企图通过九玄封天阵获取太古寒龙之力

**行为动机**:
- 追求突破飞升，但资质受限
- 发现了玄冥真人的上古传承
- 企图通过阴谋获取寒龙之力突破境界
