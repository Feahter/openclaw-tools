# TextPact - 深渊迷城

## 🚀 快速开始

### 试玩方式

**方式一：HTTP 服务器（推荐）**
```bash
cd ~/projects/textpact/www
python3 -m http.server 8080
# 浏览器访问: http://localhost:8080
```

**方式二：直接打开 HTML 文件**
```bash
open /Users/fuzhuo/.openclaw/workspace/projects/textpact/www/index.html
```

## 🎮 指令说明

### 基础指令

| 指令 | 说明 | 消耗 |
|------|------|------|
| `探索` / `look` | 查看当前房间详情 | 5 |
| `北` / `n` | 往北移动 | 3 |
| `南` / `s` | 往南移动 | 3 |
| `东` / `e` | 往东移动 | 3 |
| `西` / `w` | 往西移动 | 3 |
| `休息` / `rest` | 恢复契约力 | 0 |

### 进阶指令

| 指令 | 说明 | 消耗 |
|------|------|------|
| `收集` / `拿` | 收集房间中的物品 | 10 |
| `祈祷` / `pray` | 随机事件 | 20 |
| `创造` / `create` | 创造新房间(化石) | 30 |
| `战斗` / `调查` | 搜索敌人 | 10 |

### 战斗指令

| 指令 | 说明 | 消耗 |
|------|------|------|
| `攻击` / `打` | 普通攻击 | 0 |
| `防御` / `防` | 减少伤害 | 0 |
| `必杀` / `大招` | 高伤害技能 | 25 |
| `逃跑` / `逃` | 脱离战斗 | 5 |

## 🎯 游戏目标

1. 探索深渊迷城的各个区域
2. 收集契约碎片，增强力量
3. 击败敌人，获得奖励
4. 做出重大抉择，形成世界化石
5. 找到传说中的「光之门」

---

## 🔗 联机方式

### 导出存档
点击「💾 导出存档」按钮，下载存档文件。

### 气泡共鸣
将存档文件发给朋友，他们点击「📂 导入存档」后：
- 契约力 +20
- 获得「共鸣碎片」物品
- 继承对方的世界化石
